## 五行生克环

```
              木
             /  \
       水生/    \木生
          /      \
         水───克──→火
          ↑        │
        金生       │火生
          │        ↓
          金←──克──土
           \      /
         土生\    /金生
              土
```

**生 (外圈)**: 木→火→土→金→水→木 — `(idx+1)%5`

**克 (内星)**: 木→土→水→火→金→木 — `(idx+2)%5`

对应代码 ORDER = `['木','火','土','金','水']`：

| 发出方 | (idx+1)%5 → 生谁 | (idx+2)%5 → 克谁 |
|--------|:--:|:--:|
| 木 | 火 | 土 |
| 火 | 土 | 金 |
| 土 | 金 | 水 |
| 金 | 水 | 木 |
| 水 | 木 | 火 |
