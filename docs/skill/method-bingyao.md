# 病药法（Method: BingYao）

## 来源

《神峰通考》—「有病方为贵，无伤不是奇」

## 功能定位

蒋文正批断流程 **Step 5**。找出命局中最突出的失衡点（病），并给出克制该失衡点的元素（药）。

## 输入

```typescript
interface BingYaoInput {
  pillars: Record<string, Pillar>;     // 四柱
  dayStrength: string;                 // 日主强弱（来自扶抑法）
  dayScore: number;                    // 日主得分（来自扶抑法）
  elementScores: Record<string, number>; // 五行力量分布（来自扶抑法）
}
```

## 输出

```typescript
type BingYaoOutput = {
  yongShen: string;  // 病药用神（五行元素）
  reason: string;    // 判断依据
} | null;            // null = 无明显之病
```

## 判断逻辑

### 1. 找病

根据日主强弱，判断什么是"病"：

**身弱时：**
- 克日主元素（官杀）得分 > 5 → 病 = 官杀（克身太过）
- 泄日主元素（食伤）得分 > 5 → 病 = 食伤（泄身太过）

**身强时：**
- 生日主元素（印星）得分 > 5 → 病 = 印旺（生身太过）
- 同日主元素（比劫）得分 > 5 → 病 = 比劫旺（比劫争财）

若以上都不满足 → 病不存在（无明显之病）。

### 2. 定药

**身弱 + 官杀为病：**
- 日主有强根（dayScore ≥ 8）→ 药 = 食伤（食伤制杀）
- 日主无强根（dayScore < 8）→ 药 = 印星（印化杀生身）

**身弱 + 食伤为病：**
- 药 = 印星（印制食伤，同时生身）

**身强 + 印星为病：**
- 药 = 财星（财破印）

**身强 + 比劫为病：**
- 药 = 官杀（官杀制比劫）

### 3. 药效检查

药的得分应 ≥ 2 才有实际效果。
若药的得分 < 2 且原局该元素完全缺失 → 仍可用，但需大运补足。

### 4. 返回

- 有病有药 → `{ yongShen: 药, reason: "XX过旺为病(得分N)，取YY(药)为药。" }`
- 无明显之病 → `null`
- 病得分 < 3 → `null`（不值得专门用药）

## 示例

**陈倩怡** （己土日主，身弱，dayScore=4）：
- 病 = 七杀木（克日主，得分 19）
- 身弱无强根（dayScore=4 < 8）
- 不可用食伤制杀（金泄身更弱）
- 药 = 火（印化杀生身）
- 用神 = 火 ✓

**黄楷伟** （壬水日主，身强，dayScore=24）：
- 病 = 比劫水（得分 24 > 5）
- 身强
- 药 = 土（官杀制比劫）
- 用神 = 土 ✓

## 使用位置

在 `yongshen.ts` 的 `determineYongShen()` 中，第五步调用。在「通关法」之后、「喜忌校准」之前。

病药法是否被采用取决于优先级判断：
- 有明显之病 + 日主身弱 → 病药法可以覆盖扶抑法（特殊命例）
- 无明显之病 → 跳过
