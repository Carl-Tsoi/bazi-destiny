# 调候法（Method: TiaoHou）

## 来源

《穷通宝鉴》（QiongTong BaoJian）— 十干十二月调候用神表

## 功能定位

蒋文正批断流程 **Step 2**。判断命局是否需要气候调节，给出调候用神。

## 输入

```typescript
interface TiaoHouInput {
  dayGan: string;       // 日干: 甲~癸
  monthZhi: string;     // 月支: 子~亥
  pillars: Record<string, Pillar>;  // 四柱（用于判断杀过旺时的安全切换）
}
```

## 输出

```typescript
interface TiaoHouOutput {
  yongShen: string;     // 调候用神（五行元素: 木/火/土/金/水）
  reason: string;       // 判断依据
}
```

## 判断逻辑

### 1. 查表

查《穷通宝鉴》十干十二月调候表（`qiongtong.ts`），获取该日干在该月的：
- `primary`: 首选调候用神（天干字符）
- `secondary`: 次选用神
- `avoid`: 忌见元素
- `reason`: 古典原文

将 `primary` 天干映射为五行元素（甲→木, 丙→火, 戊→土, 庚→金, 壬→水 等）。

### 2. 安全检查：首选是否为克身元素

如果首选元素 == 克日主元素（即七杀/正官五行），需要检查克身是否过旺：
- 统计四柱中七杀数量（每透出 +3，每藏干主气 +2）
- 统计四柱中正官数量（每透出 +2，每藏干主气 +1）
- 检查地支半合是否增强克身力量（亥卯→木+2, 寅午→火+2, 巳酉→金+2, 申子→水+2）
- 若总分 ≥ 4 → 克身过旺，不能再用克身元素调候
  - 切换到 `secondary` 中的印星（化杀）元素
  - 若 `secondary` 中无印星 → 取克身元素的「生者」（印化杀：木生火→用火, 火生土→用土, 土生金→用金, 金生水→用水, 水生木→用木）

### 3. 返回

- 未触发安全检查 → 返回首选元素
- 触发安全检查 → 返回印化杀元素

## 古典依据

《穷通宝鉴》原文：
> 调候为急。冬月金寒水冷，非丙不暖。夏月火炎土燥，非水不济。

## 使用位置

在 `yongshen.ts` 的 `determineYongShen()` 中，第二步调用。在「有杀先论杀」之后、「扶抑法」之前。

调候结果是否被采用为最终用神，取决于编排器的优先级判断：
- 冬夏月 + 身弱 → 调候优先
- 春秋月或身强 → 调候降级为参考，扶抑法优先
