# 五行性质细分 — 土篇

> 土为五行之母，居中宫，性质随干湿寒暖变化最大。
> 参考：《穷通宝鉴》《子平真诠》地支藏干

## 一、干湿分类

### 湿土

| 名称 | 类型 | 藏干 | 五行偏性 |
|------|------|------|----------|
| 辰 | 水库 | 乙（木）癸（水）戊（土） | 含水，湿 |
| 丑 | 金库 | 癸（水）辛（金）己（土） | 含水，湿 |
| 己 | 天干 | — | 田园之土，润 |

### 燥土

| 名称 | 类型 | 藏干 | 五行偏性 |
|------|------|------|----------|
| 未 | 木库 | 乙（木）丁（火）己（土） | 含火，燥 |
| 戌 | 火库 | 辛（金）丁（火）戊（土） | 含火，燥 |
| 戊 | 天干 | — | 城墙之土，干 |

## 二、寒暖分类

| 条件 | 状态 | 影响 |
|------|------|------|
| 冬月（亥子丑）+ 无火 | 冻土 | 不生金 |
| 夏月（巳午未）+ 无水 | 热土 | 脆金力增强 |
| 春秋月 / 有火调候 | 暖土 | 正常运作 |

## 四、对 power-distribution.ts 的影响

### 印星（土）生日主（金）的计分

当前：所有土一视同仁，全部计入印星生身。

应改为：
- 辰、丑、己 → 计入 100% 生金分
- 未、戌、戊 → 不计入生金分（或不生金，或反脆金）
- 丑在冬月无火 → 不计入生金分（冻土）
- 未戌在夏月无水 → 脆金（金得分要扣减）

### 官杀（土）克日主（水）的计分

- 燥土（未戌戊）→ 克水力弱，应降权（×0.5）
- 湿土（辰丑己）→ 克水力强，正常计分

### 对火的影响

- 湿土晦火：辰丑在日主为火时，应计为「克火」而非单纯的食伤泄秀
- 燥土助火：未戌在火局中增强火力

## 五、对病药法的影响

当金为病时，需要区分印星土里哪些是真正的帮凶（湿暖土）、哪些不是（燥土/冻土）：
- 湿暖土生金 → 计入病的帮凶
- 燥土脆金 → 不算帮凶，甚至可能是制病的助力
