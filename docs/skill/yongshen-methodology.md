# 用神判断体系 — 蒋文正《论命琐记》批断流程

> 来源：蒋文正《论命琐记》第一、二、三集
> 原则：按批断顺序依次执行，每个方法在特定位置被调用，不互相乱给分

---

## 蒋文正批断八步（用神相关为前六步）

```
 1. 定格局   → 普通格 / 从格 / 化格
 2. 查七杀   → 有杀先论杀
 3. 病药验证 → 有病方为贵
 4. 看通关   → 两行对峙 → 通关化解
 5. 看调候   → 寒暖燥湿
 ─────────────────────────────
   喜忌校准 → 综合定用神 + 喜忌
```

后两步（岁运互动、对盘验证）属于全盘批断，不在此模块范围。

---

## 架构：yongshen.ts 是编排器

`yongshen.ts` 不包含具体判断算法。四个方法各自独立纯函数，yongshen.ts 按顺序调用：

```
yongshen.ts (编排器)
    │
    ├── 1. 格局判断  → pattern.ts (已有)
    ├── 2. 七杀检查  → method-qisha.ts (有杀先论杀)
    ├── 3. 调候法    → method-tiaohou.ts
    ├── 4. 暗拱检查  → method-angong.ts (查暗拱暗会)
    ├── 5. 扶抑法    → method-fuyi.ts
    ├── 6. 通关法    → method-tongguan.ts
    ├── 7. 病药法    → method-bingyao.ts
    │
    └── 8. 喜忌校准  → 编排器内部逻辑
```

## 每一步详解

### 第一步：定格局

> **当前范围：正格八格。从格/化格为后续 TODO。**

正格判断：由月令透干法定格局（`pattern.ts` 已有）。
- 月支藏干透出天干，且十神非比劫 → 即定为该十神格
- 无透出 → 取月支主气为格

输出：正格格局名称（正官格 / 七杀格 / 正财格 / 偏财格 / 正印格 / 偏印格 / 食神格 / 伤官格）

> **TODO**: 从格判断（日主极弱无根 + 全局克泄一方独大）和化气格判断（日干五合 + 月令支持化神）

### 第二步：有杀先论杀

> 「有杀先论杀，无杀方论财官」— 蒋文正

判断依据：
1. 原局是否有七杀？（天干透出、地支藏干主气）
2. 七杀是否有制化？
   - 食神制杀 → 化为权
   - 印星化杀 → 化杀生身
   - 无制无化 → 攻身大凶

处理：
- 七杀无制无化 + 日主身弱 → 用神必须解决七杀问题
  - 日主有强根 → 食伤制杀
  - 日主无强根 → 印星化杀
- 七杀有制化 或 无七杀 → 继续下一步

### 第三步：看调候

> 「冬月无火寒症，夏月无水燥症」— 蒋文正

判断依据：
1. 月令是否为冬夏？（亥子丑 = 冬，巳午未 = 夏）
2. 查《穷通宝鉴》十干十二月调候表

处理：
- 冬夏月 + 日主身弱 → 调候优先，用神取调候推荐
- 春秋月 → 调候仅作辅助参考
- 日主身强 → 调候降级，扶抑优先

### 第四步：查暗拱暗会

> 「丑卯拱寅、申戌拱酉」— 蒋文正

判断依据：
1. 相邻两地支是否构成拱合？（丑卯、寅辰、巳未、申戌、亥丑）
2. 拱出的地支五行是什么？
3. 原局是否缺该五行？

处理：
- 拱出五行 + 原局缺失该五行 → 可补足，该五行作为喜神候选
- 拱出五行 + 原局已有 → 增强该五行力量
- 无拱合 → 跳过

输出：拱出元素列表

### 第五步：定扶抑

判断依据（《子平真诠》《滴天髓》）：
1. 五行力量分布计算
2. 日主强弱判定（身强 / 身弱 / 中和偏旺 / 中和偏弱）

处理：
- 身强 → 抑（克泄）：食伤泄秀 或 官杀克制
- 身弱 → 扶（生扶）：印星生身 或 比劫扶助
- 中和 → 视大运走势而定

**注意**：此处的日主强弱结果同时也为步骤 2、3、6、7 提供判断依据。

### 第六步：看通关

> 「两神相峙，取通关为要」—《滴天髓》

判断依据：
1. 五行中是否有两行对峙（相克且力量接近）？
2. 对峙是否造成流通断裂？

对峙判定条件：
- 克方与被克方得分差距 ≤ 高者 × 50%
- 双方得分均 ≥ 5
- 两者之间的通关元素得分 < 3（缺失或极弱）

处理：
- 存在对峙 → 用神 = 通关元素（位于克方和被克方之间的五行）
  - 金木对峙 → 水通关
  - 木土对峙 → 火通关
  - 土水对峙 → 金通关
  - 水火对峙 → 木通关
  - 火金对峙 → 土通关
- 无对峙 → 跳过

### 第七步：病药验证

> 「有病方为贵，无伤不是奇」—《神峰通考》

注意：蒋文正在《论命琐记》中将病药法定位为**验证步骤**而非主力方法。扶抑法已有结论后，用病药法检查是否有遗漏。

判断依据：
1. 是否存在过旺且为害的元素（病）？
2. 是否存在有效克制该病的元素（药）？

处理：同 `method-bingyao.md` 中的定义。

## 最终用神确定（编排器逻辑 — 正格）

> **核心原则：扶抑为体，调候为用。**
>
> 扶抑法是基础——日主强弱决定了用神的大方向（身弱喜印比，身强喜官食财）。
> 调候法、通关法、病药法都是在扶抑框架内的微调。
> **若某方法推荐的用神落在扶抑忌神中，该方法不能作为最终用神**，仅作参考。

按蒋文正批断顺序：

```
baseXi = 扶抑法喜神集合  // 身弱→[印,比劫], 身强→[官,食,财]

if 七杀透出且无制无化:
    candidate = 制杀/化杀元素
    if candidate 在 baseXi 中 → 最终用神 = candidate
    else → 最终用神 = 扶抑结果（七杀方法被扶抑否决）

if 冬夏月 and 身弱:
    candidate = 调候元素
    if candidate 在 baseXi 中 → 最终用神 = candidate
    else → 调候降级为健康参考，最终用神 = 扶抑结果

if 有明显对峙:
    candidate = 通关元素
    if candidate 在 baseXi 中 → 最终用神 = candidate
    else → 最终用神 = 扶抑结果

if 有明显之病:
    candidate = 药
    if candidate 在 baseXi 中 → 最终用神 = candidate
    else → 最终用神 = 扶抑结果

else:
    最终用神 = 扶抑结果
```

## 喜忌推导

扶抑法胜出 → 经典规则（身弱喜印比忌官食财，身强喜官食财忌印比）
其他方法胜出 → 紧绕用神（喜 = 用神 + 生用神者，忌 = 克用神者 + 扶抑辅助）
用神必须入喜神
喜忌互不重叠
