# 第二部分：经典诗赋逻辑条件分支拆解（IF-THEN 规则库）

## 2.1 《继善篇》规则库

**规则 J01：用神不可损伤**
- IF 日主用神（财/官/印/食）被刑冲破害 → THEN 格局破败，凶

**规则 J02：日主最宜健旺**
- IF 日主身弱 → THEN 纵遇财官亦为寒士
- IF 日主太柔 → THEN 纵遇财官为寒士（反说）

**规则 J03：年伤日干，名为主本不和**
- IF 年干克制日干 → THEN 主本不和，根基有损

**规则 J04：岁月时中，大怕杀官混杂**
- IF 官杀并见于月时 → THEN 格局不纯，祸患百端

**规则 J05：官星正气，忌见刑冲**
- IF 正官遇刑冲 → THEN 官星破损，贵气减损

**规则 J06：时上偏财，怕逢兄弟（比劫）**
- IF 时上偏财 + 比肩劫财 → THEN 破财夺财

**规则 J07：生气印绶，利官运，畏见财乡**
- IF 印绶为用 + 行财运 → THEN 破印伤母、丢官卸职

**规则 J08：七杀偏官，喜制伏，不宜太过**
- IF 七杀有制 → THEN 化为权柄
- IF 七杀制伏太过 → THEN 反不为福

**规则 J09：伤官复行官运，不测灾来**
- IF 原局伤官 + 行官运 → THEN 祸患立至（伤官见官）

**规则 J10：阳刃冲合岁君，勃然祸至**
- IF 阳刃 + 流年/大运冲合 → THEN 灾祸

**规则 J11：富而且贵，定因财旺生官**
- IF 财旺生官 → THEN 富贵双全

**规则 J12：非夭则贫，必是身衰遇鬼**
- IF 身弱 + 官杀攻身 → THEN 贫夭

**规则 J13：财多身弱，正为富屋贫人**
- IF 财多 + 身弱 → THEN 富屋贫人，虽富不享

**规则 J14：以杀化权，定显寒门贵客**
- IF 七杀有制化为权 → THEN 寒门出贵客

**规则 J15：登科甲第，官星临无破之宫**
- IF 官星无破 → THEN 科举成名

**规则 J16：印绶被伤，倘若荣华不久**
- IF 印绶被伤（财来坏印） → THEN 荣华不久

**规则 J17：有官有印，无破作廊庙之材**
- IF 官印双全 + 无破 → THEN 高官厚禄

**规则 J18：无官无印，有格乃朝廷之用**
- IF 无官无印 + 入别格 → THEN 亦为朝廷所用

**规则 J19：身弱遇官，得后徒然费力**
- IF 身弱 + 遇官 → THEN 虽得官亦徒劳

**规则 J20：为人好杀，阳刃必犯于偏官**
- IF 阳刃 + 偏官（七杀） → THEN 为人好杀、刚强

**规则 J21：素食慈心，印绶遂逢于天德**
- IF 印绶 + 天德贵人 → THEN 慈悲善良

**规则 J22：身强杀浅，假杀为权**
- IF 身强 + 杀浅 → THEN 假杀为权，掌权柄

**规则 J23：杀重身轻，终身有损**
- IF 杀重 + 身轻 → THEN 终身受损

**规则 J24：衰则变官为鬼，旺则化鬼为官**
- IF 身衰遇官 → THEN 官变为鬼（灾）
- IF 身旺遇杀 → THEN 鬼化为官（权）

**规则 J25：太岁乃众杀之主，入命未必为灾；若遇战斗之乡，必主刑于本命**
- IF 太岁与命局冲克战 → THEN 灾祸

**规则 J26：岁伤日干，有祸必轻；日犯岁君，灾殃必重**
- IF 岁干克日干 → THEN 祸轻
- IF 日干克岁干（日犯岁君） → THEN 灾重
- EXCEPT 五行有救 → THEN 其年反必为财

**规则 J27：五行不可偏枯，务禀中和之气**
- Core principle: 中和为贵

## 2.2 《喜忌篇》规则库

**规则 X01：时逢七杀，见之未必为凶；月制干强，其杀反为权印**
- IF 时上七杀 + 月令制伏 + 日干强 → THEN 杀化为权印（贵格）

**规则 X02：财官印绶全备，藏蓄于月季之中**
- IF 财官印全备 + 藏于辰戌丑未月 → THEN 富贵之命

**规则 X03：官星财气长生，镇居于寅申巳亥**
- IF 财官居于长生之位（寅申巳亥） → THEN 贵格

**规则 X04：月生日干无天财，乃印绶之名**
- IF 月令生扶日干 + 无财星 → THEN 印绶格

**规则 X05：日禄居时没官星，号青云得路**
- IF 日干禄在时支 + 无官星 → THEN 归禄格，青云得路

**规则 X06：阳水叠逢辰位，是壬骑龙背之乡**
- IF 壬日 + 地支多辰 → THEN 壬骑龙背格

**规则 X07：阴木独遇子时，为六乙鼠贵之地**
- IF 乙日 + 子时 → THEN 六乙鼠贵格

**规则 X08：庚日全逢润下，忌壬癸巳午之方；时遇子申，其福减半**
- IF 庚日 + 申子辰全水局 → THEN 井栏叉格
- EXCEPT 时逢子/申 → THEN 其福减半
- 忌 壬癸巳午

**规则 X09：若逢伤官月建，如凶处未必为凶；内有正倒禄飞，忌官星，亦嫌羁绊**
- IF 月令伤官 + 伤官伤尽 → THEN 反为贵格
- EXCEPT 见官星 → THEN 祸

**规则 X10：六癸日时逢寅位，岁月怕戊己二方**
- IF 癸日 + 寅时 → THEN 刑合格
- EXCEPT 年月有戊己 → THEN 破格

**规则 X11：甲子日再逢子时，畏庚辛申酉丑午**
- IF 甲子日 + 子时 → THEN 子遥巳格
- EXCEPT 见庚辛申酉丑午 → THEN 破格

**规则 X12：辛癸日多逢丑地，不喜官星；岁时逢子巳二宫，虚名虚利**
- IF 辛/癸日 + 多丑 → THEN 丑遥巳格
- EXCEPT 子巳 → THEN 虚名虚利

**规则 X13：拱禄拱贵，填实则凶**
- IF 拱禄/拱贵格 → 忌填实（忌见正禄位地支）

**规则 X14：时上偏财，别宫忌见**
- IF 时上偏财格 → 他柱忌见偏财

**规则 X15：六辛日逢戊子，嫌午未位，运喜西方**
- IF 辛日 + 戊子时 → THEN 六阴朝阳格
- EXCEPT 午未 → THEN 破格
- 喜西方金运

**规则 X16：五行遇月支偏官，岁月时中亦宜制伏**
- IF 月支偏官（七杀） → 须要制伏

**规则 X17：类有去官留杀，亦有去杀留官**
- IF 去官留杀 → THEN 武贵
- IF 去杀留官 → THEN 文贵

**规则 X18：四柱纯杂有制，定居一品之尊；略有一位正官，官杀混杂反贱**
- IF 纯（不混杂） + 有制 → THEN 一品之尊
- IF 官杀混杂 → THEN 反贱

**规则 X19：戊日午月，勿作刃看；时岁火多，却为印绶**
- IF 戊日 + 午月 + 火多 → THEN 作印绶看，不作阳刃看

**规则 X20：月令虽逢建禄，切忌会杀为凶**
- IF 月令建禄 + 会合七杀 → THEN 凶

**规则 X21：官星七杀交差，却以合杀为贵**
- IF 官杀交差 + 合杀 → THEN 贵

**规则 X22：印绶生月，岁时忌见财星；运入财乡，却宜退身避位**
- IF 印绶格 + 财运 → THEN 退身避位

**规则 X23：劫财阳刃，切忌时逢；岁运并临，灾殃立至**
- IF 时逢阳刃/劫财 + 岁运并临 → THEN 灾殃立至

**规则 X24：十干背禄，岁时喜见财星；运至比肩，号曰背禄逐马**
- IF 背禄（伤官）+ 见财星 → THEN 吉
- IF 背禄 + 行比肩运 → THEN 背禄逐马，凶

**规则 X25：日干无气，时逢阳刃不为凶**
- IF 日干弱 + 时逢阳刃 → THEN 不为凶（阳刃帮身）

**规则 X26：官杀两停，喜者存之，憎者弃之**
- IF 官杀两停 → 制化取舍要看喜忌

**规则 X27：地支天干合多，亦云贪合忘官**
- IF 合多 → THEN 贪合忘官

**规则 X28：四柱伤官，运入官乡必破**
- IF 四柱伤官 + 行官运 → THEN 破败

## 4.2 《神趣八法》

### 类象
- 天干一类地支一片：如春生甲乙人，地支寅卯辰全
- **规则：** 忌引至死绝之乡；引至生旺为秀气加临

### 属象
- 天干甲乙木 + 地支寅卯未全

### 从象
- 日主无根，地支全一行之气：从金、从土、从水、从木、从火
- **规则：** 天干有甲己字或有根者不吉；有秀气者吉

### 化象
- 如甲乙日生辰戌丑未月，天干己字合甲，化土
- **规则：** 喜行火运；忌甲乙木生旺运
- 两个甲争合一个己 → 争合；乙字露出 → 妒合 → 破格

### 照象
- 如丙日巳午未，时上卯木 → 木火相照
- 壬癸日申子辰全，时上金 → 金水相照

### 返象
- 月令用神引至时上绝地 → 用之不用 → 返运
- **规则：** 返之太甚则不吉

### 鬼象
- 秋月生甲乙日 + 地支四金 → 鬼象
- **规则：** 喜鬼生旺运；怕死绝之乡

### 伏象
- 寅午戌三合全 + 午月生逢壬日 + 天干无丁透露 + 壬水无根
- **规则：** 运至木火之乡皆吉；水旺之乡不利

## 4.3 格局生死引用

| 格局 | 死兆条件 |
|------|---------|
| 印绶 | 见财 + 行财运 + 死绝 → 必入黄泉（柱有比肩可解） |

# 第五部分：年/月/日/时柱力量地位与地支力量对比

## 5.1 四柱力量地位定论

| 柱位 | 譬喻 | 六亲 | 功能 |
|------|------|------|------|
| 年柱 | 根 | 祖上 | 为本，知世代宗派盛衰之理，知祖业财产 |
| 月柱 | 苗 | 父母、兄弟 | 为提纲，知亲荫名利有无，月令为命之关键 |
| 日柱 | 花 | 己身、妻妾 | 日干为主，日支为妻妾之位 |
| 时柱 | 果 | 子息 | 为辅佐，知嗣续之所归，定晚年休咎 |

**核心规则（《看命入式》）：**
- 取年为根，为上祖财产，知世运之盛衰
- 取月为苗，为父母，则知亲荫之有无
- 日干为己身，日支为妻妾，则知妻妾之贤淑
- 时为花实，为子息，方知嗣续之所归
- 法分月气深浅，得令不得令

- "月为提纲，带官星印绶，则慷慨聪明、见识高人"（《论月令》）

```
  │   ├── 检查例外/救应条件
  │   └── 输出匹配到的断语
  │
  └── 5. 综合报告输出
```

## 6.3 断语分类系统

### A. 命中注定类（原局条件即成立）
- 如："年带伤官，父母不全" → 原局年柱见伤官即触发
- 如："日坐空亡，难为妻妾" → 日支落空亡即触发

### B. 大运条件类（须结合大运）
- 如："伤官复行官运，不测灾来" → 原局伤官 + 运见正官
- 如："印绶被伤，倘若荣华不久" → 大运破印

### C. 流年触发类（须结合流年）
- 如："阳刃冲合岁君，勃然祸至" → 流年与阳刃冲合
- 如："日犯岁君，灾殃必重" → 日干克流年干

### D. 复合条件类（多项条件须同时成立）
- 如："富而且贵，定因财旺生官" → 财旺 + 官旺 + 身强
- 如："非夭则贫，必是身衰遇鬼" → 身弱 + 官杀攻身

## 6.4 核心诗诀触发优先级

```
级别 1（最高优先级）：格局定义
  例：曲直格、炎上格、从革格、润下格、稼穑格
  触发条件 → 整个四柱的格局认定

级别 2：通用于各种八字的祖训断语
  例：
  - "伤官见官，为祸百端"
  - "阳刃冲合岁君，勃然祸至"
  - "官杀混杂，身弱则贫"
  - "财多身弱，富屋贫人"

级别 3：柱位特定断语
  例：
  - "年带伤官，父母不全"
  - "月带伤官，兄弟不完"
  - "日带伤官，妻妾不完"
  - "时带伤官，子息无传"
  - "月上伤官，父母不完；日上伤官，难为妻妾；时上伤官，子孙无传"

级别 4：五行特定断语
  例：
  - "金木交差刑战，仁义俱无"
  - "水火递互相伤，是非日有"
  - "金弱遇火炎之地，血疾无疑"
  - "土虚逢木旺之乡，脾伤定论"
```

## 6.5 关键技术要点

1. **条件叠加：** 同一个命局可能匹配多条断语，需要按优先级展示、并按矛盾性过滤
2. **救应规则优先：** 如原局有"七杀有制"，则"七杀无制"类的断语不触发
3. **五行转换：** 对于"火土伤官宜伤尽；金水伤官要见官"等分五行不同规则，引擎需要区分五行属性
4. **轻重权衡：** 如"伤官见官，元有者重，元无者轻"
5. **月令权重：** 月令的判断在所有规则中具有最高优先级的"提纲"地位
