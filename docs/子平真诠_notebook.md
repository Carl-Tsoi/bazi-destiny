## 一、月令定格与用神/相神定义（系统的基础变量）

### 1.1 月令取格算法 — 绝对优先级与判断流程

《子平真诠》核心原则：**八字用神，专求月令，以日干配月令地支，而生克不同，格局分焉。**

```
取格优先级（从高到低）：

1. 月令本气透干 → 直接取本气为格
   - 如寅月透甲 → 按甲木取格
   
2. 月令本气不透，支中余气透干 → 取透出之干为格
   - 如寅月不透甲而透丙 → 取丙火（食神/偏财等）为用
   
3. 月令三合会支成局 → 以会合所化之五行取格
   - 如寅月不见透干，而寅午戌三合 → 化为火局取格
   
4. 月令本气不透、余气不透、无会合 → 以月令本气为用
   
5. 月令无可用之神（建禄/月劫）→ 另寻四柱财官煞食透干会支
   - 如甲生寅卯月 → 建禄月劫格，另取他神为用
```

**特殊情况：用神变化**
- 月令所藏不止一干，透出何干即取何干为用
- 会合能改变月令性质（如寅午戌三合化火，亥卯未三合化木等）
- 变化分三类：
  - **变之而善**：忌神化为喜神（如癸生寅月，伤官化财）
  - **变之而不善**：喜神化为忌神（如丙生寅月，印绶化劫）
  - **变而不失本格**：变化后原格仍在（如辛生寅月，透丙化官又透甲，成财官双格）

### 1.2 用神、相神、忌神的严格定义

#### 用神（格局的主宰）

```
定义：月令地支中取出的、决定八字格局的核心神煞。
是整个八字的"君主"，格局之灵魂。
```

**用神分类：**

| 类别 | 包含 | 处理方式 |
|------|------|----------|
| **善用神（顺用）** | 财、官、印、食 | 宜顺其势而生之护之 |
| **不善用神（逆用）** | 煞、伤、刃、劫 | 宜逆其性而制之化之 |

**顺用原则：**
- 财喜食神以相生，生官以护财
- 官喜透财以相生，生印以护官
- 印喜官煞以相生，劫财以护印
- 食喜身旺以相生，生财以护食

**逆用原则：**
- 七煞喜食神以制伏，忌财印以资扶
- 伤官喜佩印以制伏，生财以化伤
- 阳刃喜官煞以制伏，忌官煞之俱无
- 月劫喜透官以制伏，利用财而透食以化劫

#### 相神（格局的辅弼）

```
定义：帮助用神成格、保护用神不被破坏的关键干支。
比喻：用神如君，相神如相——辅佐用神完成格局。
重要性：伤用神甚于伤身，伤相甚于伤用！
```

**相神的角色：**
1. **互补者**：官逢财生 → 官为用，财为相
2. **生产者**：财旺生官 → 财为用，官为相
3. **保护者**：煞逢食制 → 煞为用，食为相
4. **整合者**：全局赖此一字而成格者，均为相神

**相神经典案例：**
- 甲用酉官，透丁（伤官）逢壬（印）→ 合伤存官，全赖壬水之相
- 戊用子财，透甲（煞）并己（合）→ 合煞存财，全赖己土之相
- 乙用酉煞，年丁月癸，时上逢戊 → 合去癸印使丁制煞，全赖戊土之相
- 癸生亥月，透丙为财逢月劫 → 卯未会化水为木，转劫生财，全赖卯未之相

#### 忌神（格局的破坏者）

```
定义：伤害用神或相神的干支。
如官格见伤而无印护，财格见劫而无食化。
```

## 二、八个常规格局的"成、败、救应"状态机

### 2.1 正官格

```
月令取格：月令为正官（如甲生酉月、丙生子月）
```

#### 成格条件（吉）
```
IF (官逢财印) AND (无刑冲破害) → 官格成
OR (财旺生官) AND (身强) → 官格成
OR (官印双全) → 官格成
OR (身官两停) → 官格成
```

- 官逢财生 → 官为用，财为相
- 官逢印护 → 印能护官
- 透财以生之，透印以护之

#### 败格条件（凶）
```
IF (官见伤官) AND (无印护) → 官格败
OR (官煞混杂) → 官格败
OR (刑冲破害月令) → 官格败
```

#### 救应条件（修复）
```
IF (官逢伤) AND (透印) → 印解伤（伤被印制）
IF (杂煞) AND (合煞留官) → 合去七煞，官格清
IF (刑冲) AND (会合解之) → 会合解刑冲，官格复
```

#### 取运规则
- 官用财印，身轻则助身，官轻则助官
- 忌：官露逢合、杂煞、重官、地支刑冲
- 官用财：运喜印绶身旺之地，忌食伤
- 官佩印：运喜财乡，伤食反吉
- 官带伤食而用印制：喜官旺印旺，切忌财运

---

### 2.2 财格

```
月令取格：月令为财星（如甲生辰月透戊、乙生戌月）
```

#### 成格条件（吉）
```
IF (财旺生官) AND (无伤官) AND (无七煞混) → 财格成
OR (财逢食生) AND (身强) AND (略带比劫) → 财格成
OR (财格佩印) AND (财印不相碍) → 财格成
OR (财用伤官化劫) AND (财不甚旺而比强) → 财格成
```

#### 败格条件（凶）
```
IF (财轻比重) → 财格败
OR (财透七煞) AND (无制) → 财格败
OR (财印相克) AND (位置不当) → 财格败
```

#### 救应条件（修复）
```
IF (财逢劫) AND (透食化之) → 食神化劫生财
IF (财逢劫) AND (生官制之) → 官能护财
IF (财逢煞) AND (食神制煞生财) → 食制煞以生财
IF (财逢煞) AND (合煞存财) → 合去七煞保存财星
```

#### 取运规则
- 财旺生官：喜身旺印绶，不利七煞伤官
- 财用食生：财食重身轻则助身，财食轻身重则行财食
- 财格佩印：喜官乡，身弱喜印旺
- 财带伤官：财运亨，煞运不利
- 财带七煞：喜食伤身旺之方

---

### 2.3 印格（印绶格）

```
月令取格：月令为印星（如甲生子月、乙生亥月）
```

#### 成格条件（吉）
```
IF (印轻逢煞) → 煞生印，印格成
OR (官印双全) → 官印皆清，印格成
OR (身印两旺) AND (用食伤泄气) → 泄秀为贵，印格成
OR (印多逢财) AND (财透根轻) → 印格成
```

#### 败格条件（凶）
```
IF (印轻逢财) → 财破印，印格败
OR (身强印重) AND (透煞) → 偏之又偏，印格败
```

#### 救应条件（修复）
```
IF (印逢财) AND (劫财解之) → 劫制财以存印
IF (印逢财) AND (合财存印) → 合去财星保存印绶
```

#### 取运规则
- 印绶用官：官露印重，财运反吉，伤食亦利
- 印用伤食：财运反吉，行官运反灾，煞运为福
- 印用七煞：运喜伤食身旺，一见财乡凶至
- 印绶遇财：运喜劫地，官印亦亨，财乡则忌

---

### 2.4 食神格

```
月令取格：月令为食神（如甲生巳月、丙生辰月）
```

#### 成格条件（吉）
```
IF (食神生财) AND (身强食旺财透) → 食格成（大贵）
OR (食带煞) AND (无财) → 食格成
OR (弃食就煞) AND (透印) → 食格成
```

#### 败格条件（凶）
```
IF (食神逢枭) → 枭神夺食，食格败
OR (生财露煞) → 财党煞破格，食格败
```

#### 救应条件（修复）
```
IF (食逢枭) AND (就煞成格) → 弃食就煞
IF (食逢枭) AND (生财护食) → 以财制枭护食
```

#### 取运规则
- 食神生财：财食轻则行财食，财食重则喜帮身
- 食用煞印：喜印旺，切忌财乡
- 食伤带煞：喜行印绶身旺，财最忌
- 食神太旺带印：运最利财，印最忌

---

### 2.5 七煞格（偏官格）

```
月令取格：月令为七煞（如甲生申月透庚、乙生酉月透辛）
```

#### 成格条件（吉）
```
IF (身强七煞逢制) → 煞格成（大贵）
OR (煞用食制) AND (煞旺食强身健) → 极高之贵
OR (七煞用印) AND (印有情通月令) → 煞格成
```

#### 败格条件（凶）
```
IF (七煞逢财) AND (无制) → 财党煞，煞格败
```

#### 救应条件（修复）
```
IF (煞逢食制) AND (印来护煞) → 逢财去印存食，煞格复
OR (煞逢财党) AND (食神制煞生财) → 食制煞兼生财
```

#### 取运规则
- 煞用食制：煞重食轻助食，煞轻食重助煞
- 煞用印绶：不利财乡，伤官为美
- 七煞用财：不利劫财，伤食皆吉
- 煞带正官：身轻喜助身，食轻喜助食

---

### 2.6 伤官格

```
月令取格：月令为伤官（如甲生午月、乙生巳月）
```

#### 成格条件（吉）
```
IF (伤官生财) AND (身强有根) → 伤官格成
OR (伤官佩印) AND (伤官旺印有根) → 伤官格成
OR (伤官旺身弱) AND (透煞印) → 煞印相生，伤官格成
OR (伤官带煞) AND (无财) → 伤官格成
```

#### 败格条件（凶）
```
IF (伤官非金水) AND (见官) → 伤官见官祸百端
OR (伤官生财带煞) → 混杂不清
OR (佩印伤轻身旺) → 印重无益
```

#### 救应条件（修复）
```
IF (伤官生财透煞) AND (煞逢合) → 合煞，财不受害
IF (伤官见官) AND (财官位置合宜) → 伤官生财来生官，反解
```

#### 取运规则
- 伤官用财：财旺身轻利印比，身强财浅喜财运
- 伤官佩印：行官煞运为宜，印运亦吉，财地则凶
- 伤官带煞：喜印忌财
- 伤官用官：运喜财印，不利食伤

---

### 2.7 阳刃格

```
月令取格：月令为阳刃（五阳有之——甲卯、丙午、戊午、庚酉、壬子）
```

#### 成格条件（吉）
```
IF (阳刃透官煞) AND (露财印) AND (不见伤官) → 阳刃格成
```

#### 败格条件（凶）
```
IF (阳刃无官煞) → 刃无制，刃格败
```

#### 救应条件（修复）
```
IF (阳刃用官煞) AND (带伤食) → 重印护之，格复
```

#### 取运规则
- 阳刃用官：运喜助官
- 阳刃用煞：煞不甚旺喜助煞，煞太重喜身旺印绶
- 阳刃官煞并出：运喜制伏，身旺亦利，财地官乡反不吉

---

### 2.8 建禄月劫格

```
月令取格：月令为禄（甲寅、乙卯）或月劫（甲卯、乙寅）
```

#### 成格条件（吉）
```
IF (透官逢财印) → 建禄月劫格成
OR (透财逢食伤) → 建禄月劫格成
OR (透煞遇制伏) → 建禄月劫格成
```

#### 败格条件（凶）
```
IF (无财官) AND (透煞印) → 建禄月劫格败
```

#### 救应条件（修复）
```
IF (建禄用官) AND (遇伤) → 伤被合，格复
IF (建禄用财带煞) AND (煞被合) → 煞被合去，格复
```

#### 取运规则
- 禄劫用官：印护者喜财，畏七煞之乘
- 禄劫用财带伤食：财食重喜印绶，财食轻宜助财
- 禄劫用煞食制：食重煞轻助煞，食轻煞重助食
- 禄劫用伤食：财运最宜，煞亦不忌，行印非吉

## 三、用神纯杂与格局高低判定

### 3.1 纯与杂的量化标准

#### 纯（吉）
```
条件：互用而两相得 → 干支配置和谐，相辅相成
```

**纯的经典案例：**
| 案例 | 配置 | 结论 |
|------|------|------|
| 辛生寅月，甲丙并透 | 财与官相生 | 纯 |
| 戊生申月，庚壬并透 | 财与食相生 | 纯 |
| 癸生未月，乙己并透 | 煞与食相克而得当 | 纯 |

#### 杂（凶）
```
条件：互用而两不相谋 → 干支配置冲突，互相破坏
```

**杂的经典案例：**
| 案例 | 配置 | 结论 |
|------|------|------|
| 壬生未月，乙己并透 | 官与伤相克 | 杂（官伤混杂） |
| 甲生辰月，戊壬并透 | 印与财相克 | 杂（财印相战） |

**纯杂转化规则：** 纯杂之理不出变化。通过干支的透出、会合、制化，可以使杂变纯，也可使纯变杂。

### 3.2 格局高低的四大决定因素

#### A. 有情 vs 无情
```
有情 = 用神与相神之间配合默契，互相增益
无情 = 用神与相神之间互相妨碍，或各不相谋
```

**有情等级（由高到低）：**
1. 有情而兼有力 → 最高格
2. 有情但非情之至 → 次高格
3. 无情 → 低格

**示例：**
- 甲用酉官，透丁（伤）逢壬（印）→ 有情（壬合丁存官）
- 甲用酉官，透丁逢癸（印）→ 有情但非情之至（癸克丁不如壬合丁）

#### B. 有力 vs 无力
```
有力 = 用神或相神根深，在支中有强根
无力 = 用神或相神根浅，无通根之地
```

**示例：**
- 乙生酉月，辛金透出，丁火刚健，秋木盛 → 三者皆备，极等之贵（有力）
- 煞强食旺而身无根 → 夭或贫（无力）

#### C. 清纯 vs 混杂
```
清纯 = 用神单一不杂，或虽多而互补
混杂 = 官煞混杂、财印混杂、伤官见官等
```

**常见混杂情况：**
- 财带食神又透官 → 不纯（官食混杂于财格）
- 煞用食制又逢财 → 格低（财生煞破食制）
- 官煞混杂 → 需要合煞留官或合官留煞

#### D. 调候配合
```
即使格局优美，若气候不宜，也难大贵。
如：冬木逢水（金寒水冷）、夏木逢火（炎燥）等，
必须考虑调候——寒需暖、热需凉。
```

### 3.3 因成得败、因败得成（格局反转机制）

#### 因成得败
```
原局已成格，因某字加入（岁运配置）而转败。
```

**经典案例：**
- 辛生亥月，透丁为用，卯未会财 → 本已成格，然卯未会财党煞 → 因成得败
- 癸生申月，秋金重重，略带财以损太过 → 本已成格，逢煞则煞印忌财 → 因成得败

#### 因败得成
```
原局本败格，因某字出现（救应）而转成。
```

**经典案例：**
- 辛生戊月，年丙时壬 → 官印逢伤本为败格，然壬不能越戊克丙，反能泄身为秀 → 因败得成
- 庚生酉月，年丙月丁时上逢壬 → 煞刃逢食本为败格，然食神合官留煞，官煞不杂 → 因败得成

## 四、十干得时不旺失时不弱（核心理论）

### 4.1 原文精要

> **"得时俱为旺论，失时便作衰看，虽是至理，亦死法也。然亦可活看。"**

### 4.2 旺与衰、强与弱的严格区分

《子平真诠》对此有四个维度的严格区分：

```
旺衰 ≠ 强弱

得时 = 旺    失时 = 衰
党众 = 强    助寡 = 弱

旺而弱者：当令但因被泄/被克而气势不强
衰而强者：失令但因有根有助而气势不衰
```

#### 分类矩阵

| 状态 | 含义 | 示例 |
|------|------|------|
| 旺而强 | 当令且有党众 | 甲乙木生寅卯月，干支多木 |
| 旺而弱 | 当令但被泄/克 | 甲乙木生寅卯月，干透丙丁，支巳午多，泄气太重 |
| 衰而强 | 失令但有根有助 | 甲乙木生申酉月，比印重叠，通根地支 |
| 衰而弱 | 失令且无助 | 甲乙木生申酉月，干支无助 |

### 4.3 根的重要性（核心机制）

**根的种类与权重：**

```
权重排序（从高到低）：
1. 长生禄旺（根之重者）—— 得一支强于三比肩
2. 墓库余气（根之轻者）—— 得一余气强于二比肩
3. 比肩（根之最轻）—— 需多比肩方成势

比劫如朋友相扶，通根如室家之可住
干多不如根重
```

#### 阳干阴干的根性差异

```
阳干：
- 长生有力（如甲木长生在亥）
- 逢库有根（如甲逢未库）

阴干：
- 阴长生不甚有力（如乙木长生在午，但力量弱）
- 逢库无用（如乙逢戌，戌中无藏木；丁逢丑，丑中无藏火）
  → 阳大阴小，阳得兼阴，阴不能兼阳
```

### 4.4 十干得时不旺失时不弱的实际应用

**核心逻辑：判断日主强弱不能只看月令是否为禄旺，必须全局评估：**

```
日主强弱判断流程：

1. 看月令 → 是否得时（旺/衰）
2. 看四柱 → 是否有根（长生禄旺 > 墓库余气 > 比肩）
3. 看党众 → 比劫印绶数量
4. 看被泄 → 是否食伤多而泄气
5. 看被克 → 是否官煞多而制身
6. 综合定论 → 全局通盘考虑，不执一隅
```

### 4.5 对陈葆欣问题的直接指向

此理论在书中第六篇明确阐述，其结论是：

> **"十干不论月令休囚，只要四柱有根，便能受财官食神而当伤官七煞。"**

这意味着：
- 日主虽失令（月令休囚），但四柱有根，仍可胜任财官
- 日主虽当令，但被泄太甚或被克太重，反不为强
- 用神的选择必须基于全局，不能仅因月令为某神就认定其为用

## 五、用神纯杂对命格高低的影响

### 5.1 用神纯杂的核心公式

```
纯 = 一字发力，全局皆活
杂 = 两神相争，彼此牵制

清者贵，混者贱
一字之差，千钧之力
半字之变，全局之美尽失
```

### 5.2 系统过滤规则

#### A. 财印不两用（除非位置互不相碍）

```
财印相克，不可并用。
例外条件：
  IF (财印位置相隔) AND (中隔比劫或官星) → 不碍
  → 如年干印、时干财，中隔比劫 → 可两用
  → 否则贪财坏印
```

#### B. 官煞不能混杂

```
官煞必须取清：
  IF (官多煞杂) AND (合煞留官) → 清
  IF (官多煞杂) AND (合官留煞) → 清
  → 留一清一，不混则贵
```

#### C. 食神与枭神不能两见（除非有救应）

```
食神逢枭 → 夺食
救应条件：
  IF (就煞成格) → 弃食就煞
  IF (生财护食) → 以财制枭
```

#### D. 伤官与正官不能并见（除非特殊情况）

```
伤官见官 → 祸百端
例外：
  - 金水伤官（金生冬月）：见官反为秀气（调候需要）
  - 伤官生财再生官：伤官间接生官，可解
```

### 5.3 纯杂渐变图谱

```
最纯 ←——————————————————————————→ 最杂
  |         |             |              |
单用一神  用神+相神   用神+忌神     多神混杂
(极贵)     (互补)      (有救应)     (无救应，下格)
```

### 5.4 格局高低判定矩阵

| 有情 | 有力 | 清纯 | 调候 | 结果 |
|------|------|------|------|------|
| ✓ | ✓ | ✓ | ✓ | 极贵 |
| ✓ | ✓ | ✓ | ✗ | 贵但不全 |
| ✓ | ✓ | ✗ | — | 中格 |
| ✓ | ✗ | — | — | 小贵 |
| ✗ | — | — | — | 贫贱 |

## 六、岁运（大运流年）与格局的转换规则

### 6.1 运助格成

```
原局格局有缺，行运补足：

IF (丁生辰月，透壬为官) AND (运逢申子会之) → 运成官格
IF (乙生辰月，申子会印) AND (运逢壬癸透之) → 运成印格
```

### 6.2 运破格局

```
行运破坏原局格局：

IF (正官无印) AND (运行伤官) → 运破官格
IF (七煞食制) AND (运逢枭神) → 枭夺食，煞无制，运破
IF (伤官佩印) AND (运行财运) → 财破印，运破
IF (阳刃用煞) AND (运逢食神) → 食制煞，刃无制，运破
```

### 6.3 因运改变格局

```
运支与原局会合，导致月令格局发生质变：

IF (原局丁生辰月，透壬为官)
   AND (运逢戌) → 透出辰中伤官 → 官格变伤官格

IF (原局壬生戌月，丁己并透)
   AND (运逢寅午会局) → 财旺生官
   AND (更逢戌运) → 变七煞格

IF (原局壬生亥月，透己用官)
   AND (运逢卯未会亥) → 建禄化伤官格
```

### 6.4 运干与运支的差异

```
干主天，动而有为
支主地，静以待用

规则：
1. 有行干而不行支者（如丙生子月亥年，逢丙丁帮身，逢巳午则冲）
2. 有行支而不行干者（如甲生酉月，辛金透官弱，逢申酉官植根，逢庚辛混煞）
3. 干同一类而不两行者（如丁生亥月，年透壬官，逢丙帮身，逢丁合官）
4. 支同一类而不两行者（如戊生卯月丑年，逢申自坐长生，逢酉会丑伤官）
```

### 6.5 大运与原局冲合的动态判定

```
冲分缓急：
- 冲年月则急
- 冲日时则缓

冲分轻重：
- 运本美而逢冲则轻
- 运既忌而又冲则重

会合可解刑冲：
- 卯冲酉，有戌合卯或辰合酉 → 不冲
- 卯刑子，有戌合卯或丑合子 → 不刑

冲亦可解冲：
- 丙生子月，卯以刑子，又逢酉 → 卯与酉冲不刑月令之官
```

## 七、面向软件格局引擎的设计建议

### 7.1 核心领域模型

基于《子平真诠》的高度公式化特点，建议采用以下面向对象抽象：

```
class BaZiPillar:
    - heavenly_stem: Stem      # 天干
    - earthly_branch: Branch   # 地支
    - hidden_stems: List[Stem] # 藏干

class BaZiChart:
    - year: BaZiPillar
    - month: BaZiPillar       # 月令（格局的核心）
    - day: BaZiPillar         # 日主
    - hour: BaZiPillar
    + get_month_orders()      # 月令司事
    + get_hidden_stems()      # 取藏干

class Pattern:                # 格（格局）
    - type: PatternType       # 官/财/印/食/煞/伤/刃/劫
    - using_god: God          # 用神
    - assisting_god: God      # 相神
    - jealous_god: God        # 忌神
    - status: PatternStatus   # 成/败/待救
    + evaluate()              # 评估格局成败
    + find_remedy()           # 寻找救应

class TransitionRule:         # 变化规则
    - condition: str          # 触发条件（透干、会支等）
    - from_elements: List     # 原五行/十神
    - to_elements: List       # 变后五行/十神
    - is_beneficial: bool     # 是否变之而善

class RemedyRule:             # 救应规则
    - broken_condition: str   # 败格条件
    - remedy_action: str      # 救应动作
    - required_elements: List # 所需干支条件

class LuckPillar:             # 大运/流年
    - stem: Stem
    - branch: Branch
    + interact_with_chart()   # 与原局交互
```

### 7.2 格局引擎核心流程（状态机）

```
输入：八字四柱 + 大运/流年
输出：格局类型、成败状态、救应建议

Step 1: 月令取格 ← 算法：
  a) 检查月令本气是否透干 → 是则取本气为用
  b) 检查余气是否透干 → 是则取透出者为用
  c) 检查三合会支 → 是则以所化之行为用
  d) 检查是否为建禄月劫 → 是则另寻他神
  e) 均无 → 以月令本气为用

Step 2: 相神配置 ← 根据格局类型，匹配标准相神

Step 3: 成败判定 ← 状态机
  - 检查成格条件列表
  - 检查败格条件列表
  - 输出当前格局状态

Step 4: 救应检查 ← 当格局为"败"时
  - 遍历救应规则库
  - 检查是否有匹配的救应条件
  - 输出救应方案

Step 5: 纯杂判定 ← 评估格局质量
  - 检查用神/相神是否被其他神煞干扰
  - 计算干扰因子数量与严重程度
  - 输出格局"清"或"混"及等级

Step 6: 岁运交互 ← 动态模拟
  - 大运干支与原局作用
  - 检查是否"运助格成"
  - 检查是否"运破格局"
  - 检查是否"因运改变格局类型"
```

### 7.3 规则引擎设计要点

```
1. 优先级系统：
   - 透干 > 会支 > 本气
   - 合 > 冲 > 刑
   - 有救 > 无救
   - 有情有力 > 有情无力 > 无情有力 > 无情无力

2. 规则分类：
   - 取格规则（月令定格局）
   - 配置规则（用神/相神/忌神匹配）
   - 成败规则（成格条件/败格条件检查表）
   - 救应规则（败格后的修复路径）
   - 变化规则（透干/会合导致的用神变化）
   - 纯杂规则（清与混的判定）
   - 岁运规则（大运流年的动态影响）

3. 特殊处理：
   - 建禄月劫格的特殊性（月令本身非用神）
   - 杂气取用的复杂性（四墓库的多元藏干）
   - 外格的处理（月令无用神时的备选方案）
   - 调候规则的叠加（气候对格局的影响修正）
```

### 7.4 "十干得时不旺失时不弱"在引擎中的实现

IS_STRONG(chart):
    // 不只看月令，通盘评估
    root_strength = calculate_root_strength(day_master, chart)
    support_count = count_support_elements(day_master, chart)
