## 1. 病药说 — 全书核心理论

> **原文出处**: 病药说类（Line 38-39）

### 1.1 病药定义

**何为病？**
> 原八字中原所害之神也。

**何为药？**
> 如八字原有所害之字，而得一字以去之谓了，如朱子所谓各因其病而药之也。

### 1.2 核心口诀

> **有病方为贵，无伤不是奇；格中如去病，财禄两相随。**

这是全书最核心的判断法则。张楠自述此前"屡以中和而究人之造化，十无一二有验"，悟得病药说后"尝八九而得其造化所以然之妙"。

### 1.3 病药对应关系表

| 八字中的病（害神） | 对应的药（去病之神） |
|---|---|
| 四柱纯土 / 杀重身轻 | 木（去土病） |
| 土厚埋金 | 木 |
| 比肩太重 | 木 |
| 用财见比肩 | 官/杀（去比肩） |
| 用食神伤官见印 | 财（去印） |

### 1.4 病药判断步骤

张楠给出了极其清晰的操作流程：

1. **看日干**
2. **次看月令** — 月令中支所属
3. **汇集同元素** — 将月令、年、月、时上的同类元素（如"火"）汇合一处
4. **判别病否** — 判断此为病或非病
5. **从重者论** — "若再看别物，由混杂不明，故曰从重者论。此理是看命下手法处。"
6. **对症下药** — 财官印绶有病则医财官印绶，身主有病则医身主
7. **中和为度** — "若病而药，大富贵之人也"

### 1.5 极端案例的哲学

> 若今之至富至贵之人，必先"劳其筋骨，饿其体肤，空乏其身，然后动心忍性，增益其所不能"，人命之妙，其犹此乎！

—— 至富至贵之人，八字必有重病，亦必有重药。

### 1.6 病药的适用范围

> 若八字纯然不旺不弱，原财官印俱无损伤，日干之气，又得中和，并无起发可观，此是平常人也。

—— **中和无病 = 平常人**。命书以病药为第一紧要。

## 2. 雕枯旺弱四病说

> **原文出处**: 雕枯旺弱四病说类（Line 40-44）

张楠提出四类"病"：

### 2.1 雕 — 雕琢之病

**概念**: 如玉石需雕琢，金需锻炼。八字若纯然无杂（如见官星无伤官、见财星无比劫、见印绶无财星、见食神无印绶），就如未雕之玉、未炼之金。

**核心**: 八字贵在"有病"，即五行之间需要有克制、雕琢，才能成器。

> 人之造化穷爱寿夭之理，亦贵宜有去留舒配，以取用焉。是以八字贵有雕也。

### 2.2 枯 — 枯萎之病

**概念**: 如木经风霜、旱苗得雨，需要先枯后方能勃发。官星、财星先有"枯"（无根），后行旺地方贵。

**核心判断**:

| 状态 | 判定 |
|---|---|
| 官贵有枯，行官旺地 | 贵不可言 |
| 财贵有枯，行财旺乡 | 财难计数 |
| 官星无根（枯而无根） | 不为贵，反为病 |
| 财星无根（枯而无根） | 不为富，反为病 |

> 贵有根而枯之病也。若或无根，而自为之枯焉，则亦非矣。

### 2.3 旺 — 旺盛之病

**概念**: 五行过旺也为病。木太旺宜行比劫运以去财星，印太旺宜行财运以破印，日干太旺宜行官杀运以制日主。

> 是以八字贵有旺之病也。

各类旺病对应的药：

| 旺病类型 | 药（解决方式） |
|---|---|
| 比劫太旺 | 行官杀运制比劫，去财星 |
| 印星太旺 | 行财运破印 |
| 日干太旺 | 行官杀运制日主 |
| 春林旺见水多 | 水益壮其神（不算病） |
| 夏月火炎见木多 | 木愈资其烈（不算病） |

### 2.4 弱 — 衰弱之病

**概念**: 过弱也是病，但弱中有生机。"六阳之弱，不至于终弱，而有《临》《泰》之可乘。六阴之弱，不至于终弱，而有《遁》《比》之可托。"

**根的重要性**:
> 弱而有根，则官星虽弱而可致其旺，财星虽弱而可致其强。

| 弱病类型 | 药 |
|---|---|
| 官星太弱 | 行官旺之乡 |
| 财星太弱 | 行财旺之地 |
| 日主太弱 | 行身旺之地 |

**核心警戒**: "犹畏弱之无根，所谓根在苗先也。"

## 3. 损益生长四药说

> **原文出处**: 损益生长四药说类（Line 45-50）

四类"药"对治四类"病"：

### 3.1 损 — 损其有余

**概念**: 对治"旺"病。木旺用金制，金旺用火克，官星有余则损官星，财星有余则损财星。

> 譬若人身元太旺为族，当以凉剂通药以济之也。

### 3.2 益 — 益其不及

**概念**: 对治"弱"病。木不及行水运滋根本或行木运茂枝叶，水不及行金运浚源流或行水运广澎湃。

> 譬若人身血气之不足，则用温药之剂以补之也。

| 不及类型 | 药 |
|---|---|
| 木气本衰，庚辛来克 | 行水运滋根本，或行木运茂枝叶 |
| 水气本衰，戊己来克 | 行金运浚源流，或行水运广澎湃 |
| 官星不足 | 喜官旺之乡 |
| 财星不足 | 喜行财旺之地 |

### 3.3 生 — 生其根基

**概念**: 对治根气不足。强调"长生"与"真生"的区别：

- **六阳生处，真为生**：甲木生亥（亥有壬水滋甲木）
- **六阴生处，俱为弱**：乙木生于午（午有丁火泄木精英）
- **六阴死处俱为生**：乙木死于亥（亥有壬水反来滋木）
- **六阳死处真为死**：甲木死于午（午中丁火泄木真精）

> 凡气之不足，故贵济之有生之药也。

### 3.4 长 — 长其气势

**概念**: 对治"当权得位"但尚未成势的五行。"生者长初，长者生之继也。"

**关键差异**: 生长与帝旺的运喜不同：

| 位置 | 木财官 | 火财官 |
|---|---|---|
| 长生（初生） | 喜行水运生之 | 喜行木运生之 |
| 帝旺（寅卯辰/巳午未） | 贵行金运克之 | 贵行水运克之 |

> 生长二字，衰旺之不同，故运行有喜生喜克之异。

## 4. 月令与得令失令

### 4.1 月令为用神

> **原文**: 月令为用神，岁时为辅佐。（Line 6）

> **原文（看命人式）**: 法分月气浅深，得令不得令，年时露出财官，须要身旺。（Line 1305）

### 4.2 得令的核心判定

**月令提纲** 是全命局的最高权重判断依据：

> 凡看命，排下八字，以日为主，取年为根，为上祖，才知世派之盛衰。取月为苗，为父母，则知亲荫之有无。日干为身，支为妻，时为花实为子息，方知嗣续之所归。法分月气浅深，得令不得令，年时露出财官，须要身旺。如身衰财旺，多反破财伤身，身旺财多财亦旺，财多称意，若无财官，次看印绶，得何局势，吉凶断之。（Line 1305）

**得令不得令的判断要点**:

1. **月气浅深** — 节气前后的深浅程度
2. **日干与月令的关系** — 月令是否为日干的旺相之地（如甲乙生寅卯月为得令）
3. **得令者身旺** — "身主要强，月提得令"（四言独步）
4. **失令者身弱** — "凡五行失令者，纵然归禄得垣，被比局持势于月令，不作身旺格矣"（Line 2529）

### 4.3 月令与正官

> 月令提纲之官，若我之本府太守本县之令尹也。但当服其管，则岂能用之。故凡月上官星，世无用官之理。（Line 52）

张楠认为月令正官如同地方长官，只能服从管束，不能当作用神使用。正官要取用时柱或年柱上的官星。

### 4.4 得令与失令的实例区分

**例 — 阴木失令**（Line 2528-2530）:

> 阴木归垣失令，终为身弱。凡五行失令者，纵然归禄得垣，被比局持势于月令，不作身旺格矣。

举例：乙卯日主，坐卯（禄），但生于寅月，甲木得令，乙木（阴木）反弱。

### 4.5 月令的辅助参考

1. 月为苗、为父母，知亲荫之有无（Line 1305）
2. 月令有用神，得父母力（Line 1307）
3. 月为提纲，带官星印绶则慷慨聪明、见识高人

## 5. 格局体系

张楠在本书中系统论述了多种格局。每种格局的核心都是"病药"思维。

### 5.1 正官格

> **原文出处**: 正官格（Line 51-98）

**定义**: 阳见阴、阴见阳（一阴一阳之谓道），如人之一夫一妇。

**月令正官的处理**:

| 情况 | 处理 |
|---|---|
| 月上官星一点 | 日主又旺 → 官星轻而日主弱 → 运行官旺最为奇 |
| 官星太旺 | 日主弱 → 官变为杀 → 杀重身轻 |

**官印双全**: 官生印、印生身，乃廊庙栋梁之大材。

**正气官星的条件**:

| 条件 | 结果 |
|---|---|
| 月令正官 + 无冲刑破害 | 功名显达 |
| 中年岁运遇财星印绶 | 台阁可登 |
| 逢官入墓 | 父死他乡土 |

**关键断语**:
> 正气官星月上推，无冲无破始为奇。中年岁运来相助，将相公侯总可为。

### 5.2 偏官格（七杀）附弃命从杀格

> **原文出处**: 偏官格附弃命从杀格（Line 98-200）

**定义**: 阳见阳、阴见阴，原非阴阳配合。

**杀与偏官的区别**:

| 状态 | 名称 | 含义 |
|---|---|---|
| 无制 | 七杀 | 克身之刀剑，凶神 |
| 有制 | 偏官 | 化为权贵，为我之奴仆 |

**七杀处置原则**:
> 凡看命先看七杀，若有七杀，就要将此七杀处置了，方能用得别物。

**制杀三层次**:

| 制伏程度 | 结果 |
|---|---|
| 制伏太过（擒制太过） | 尽法无民，奴仆力衰，不能为我运动 |
| 制伏不及 | 奴欺主矣 |
| 制伏中和 | 化为权贵 |

**偏官格的核心断语**:

| 条件 | 结论 |
|---|---|
| 偏官有制化为权 + 岁运行身旺地 | 功名大用福双全 |
| 一位偏官，制伏有二三 | 复行制伏运反不作福（尽法无民） |
| 杀强有制 | 英俊文章发少年 |
| 运衰弱，欲其大用也难 | 然 |
| 从杀多是大富大贵之人 | "平生为富贵，只因杀重身柔" |
| 身弱杀重无制 | 虎加翼，反为所噬 |

**七杀制伏的关键规律**:

> 杀不离印，印不离杀。杀印相生，功名显达。（Line 1766）
> 身强杀浅，不宜有制。（Line 1734）

### 5.3 月支正财格 附弃命从财格

> **原文出处**: 月支正财格 附弃从财格（Line 310-340）

**核心条件**:

| 条件 | 结论 |
|---|---|
| 身主有气（旺） | 能任财，财为养命之源 |
| 身弱财多 | 不能任，如盗偷财物，反为害命之物 |
| 日干旺 + 比劫多 | 喜官杀去比劫，存起财星 |
| 身弱财多 + 再见官杀 | 性命不可保 |

**关键断语**:

> 逢财喜杀而遇杀，十有九贵。

### 5.4 伤官食神格

> **原文出处**: 伤官食神格（Line 361-400）

**伤官本质**: 伤杀其官，不服官管，如弑杀上官之类。

**伤官分真假**:

| 类型 | 条件 | 喜忌 |
|---|---|---|
| 真伤官 | 甲乙生于巳午未月 | 喜北方水运破伤官，扶木气 |
| 假伤官 | 甲乙生于正二月见火 | 喜南方火运泄精英，金主富贵 |

**伤官关键断语**:

| 条件 | 结论 |
|---|---|
| 伤官伤尽，四柱不留一点官星，行身旺及印运 | 贵 |
| 伤官见官，为祸百端 | 凶 |
| 伤官见印绶 | 贵不可言 |
| 伤官无财可恃 | 虽巧必贫 |
| 火土伤官 | 宜伤尽 |
| 金水伤官 | 喜见官 |
| 木火伤官 | 官要旺 |
| 土金官去 | 反成官 |
| 惟有水木伤官格 | 财官两见反为欢 |

### 5.5 印绶格

> **原文出处**: 印绶格（Line 481+）

**印绶核心**:
> 印绶多而宜见杀，伤官重而不忌官。（Line 1735）

### 5.6 阳刃格 附比劫、建禄格

> **原文出处**: 阳刃格 附比劫、建禄格（Line 536+）

**阳刃本质**: 刃乃身之恶物，大怕身旺逢之。

**处理原则**: 得一重杀与刃相匐，化为权星。若见官与刃冲战，乃成恶性杀。

## 6. 特殊格局

### 6.1 曲直仁寿格（木）

> **原文出处**: 曲直仁寿格（Line 761-769）

**条件**: 甲日干 + 地支寅卯辰全（或亥卯未全）

**核心规定**:
- **最忌**: 庚申辛酉冲破东方秀气（虽夭亦贵）
- **不忌**: 寅卯辰字太多，及壬癸生木类
- **必要条件**: 寅卯辰三字俱全
- **行运**: 喜东北方，怕西方运

**断语**: 亥卯未逢于甲乙，富贵无疑。木传寅卯辰之方，功名自有。

**注意**: 张楠特别指出 — "有一甲申日干者反贫，盖为有申金，能破东方秀气。"

### 6.2 稼穑格（土）

> **原文出处**: 稼穑格（Line 770-779）

**条件**: 戊己日干 + 辰戌丑未全 + 四柱无官杀

**核心规定**:
- **适用月份**: 丑辰戌月（土弱方作此格）；未月土太旺则不入此格
- **行运**: 喜南方火土之地及西方金制木之运
- **最忌**: 木运（破稼穑必死）

**断语**: 戊己局全四季，荣冠诸曹。

### 6.3 炎上格（火）

> **原文出处**: 炎上格（Line 780-785）

**条件**: 丙丁日 + 寅午戌全（或巳午未全）

**核心规定**:
- **要求**: 火虚有焰（不宜太炎）
- **最畏**: 水破格
- **行运**: 喜东方木运
- **断语**: 寅午戌遇于丙丁，荣华有日。火临巳午未之域，显达之人。

### 6.4 润下格（水）

> **原文出处**: 润下格（Line 786-790）

**条件**: 壬癸日 + 申子辰全（或亥子丑全）

**核心规定**:
- **忌**: 辰戌丑未官乡
- **喜**: 西方运，不宜东南
- **断语**: 壬癸格得申子辰，福优财足。水归亥子丑之源，荣华之客。

### 6.5 从革格（金）

> **原文出处**: 从革格（Line 791-795）

**条件**: 庚辛日 + 巳酉丑全（或申酉戌全）

**核心规定**:
- **忌**: 南方火运
- **喜**: 庚辛旺运
- **断语**: 庚辛局全巳酉丑，位重权高。金备申酉戌之地，富贵无亏。

**张楠评注**: 此格与润下格"吾见多矣，未尝有富贵者"。止有曲直、稼穑二格多富贵，火全巳午未格亦见其美。

### 6.6 从化格

> **原文出处**: 从化格（Line 808-831）

**核心原则**:

| 化气组合 | 化神 |
|---|---|
| 甲己 | 化土 |
| 乙庚 | 化金 |
| 丙辛 | 化水 |
| 丁壬 | 化木 |
| 戊癸 | 化火 |

**关键条件**:

1. **六阴日主**身弱多作从化，多主富贵
2. **六阳日干**不能从化
3. 从化成功后行化神禄旺之运则发，行禄绝之运则死

**断语**:
> 古人论造，先论从化，从化不成，方论财官。财官无取，方论格局。若从化成局，则富贵备矣。

> 化之真者，名公巨卿；化之假者，孤儿异性。

> 不化不从，淹留仁路之人。得化得从，显达功名之士。化志禄旺者生，化成禄绝者死。

### 6.7 弃命从杀格

> **原文出处**: 偏官格附弃命从杀格（Line 100, 1785-1792）

**条件**: 日主全无一点生气 + 四柱纯然有官杀

**核心原则**:

| 条件 | 结论 |
|---|---|
| 日主无根从杀 | 行财杀运则富贵 |
| 见八字有根处/制杀运 | 如从盗又思归父母兄弟之乡，必招祸 |
| 六阴日干 | 有从之之理（如妇人属阴从人） |
| 六阳日干见杀多 | 只作杀重身轻看，若日主全无气亦作弃命看 |
| 从杀者必然富贵 | 但多夭 |

**口诀**:
> 弃命从杀，须要会杀。从财忌杀，从杀喜财。若逢根气，命殒无猜。

### 6.8 弃命从财格

> **原文出处**: 月支正财格附弃从财格（Line 310-319, 1791-1792）

**条件**: 不论阴阳日主，身主无力不能任其财，只得舍命而从之。

**核心原则**:

| 条件 | 结论 |
|---|---|
| 从财成功 | 行财运则富贵 |
| 畏身入财、印生之地 | 同弃命从杀而论 |
| 从财忌杀 | 见杀破格 |
| 若逢根气 | 命殒无猜 |

**示例**: 阴火酉月，弃命就财，北方入格，南地为灾。

## 7. 格局高低与病药关系

### 7.1 病药成就格局高低

**最高层次 — 大病大药**:
> 若病而药，大富贵之人也。（Line 39）

**中等层次 — 有病有药但轻重失调**:
> 本身病重而得药重，又宜行运，以取其中和。（Line 39）

**次等层次 — 病药得当但格局不纯**:
> 如八字纯然不旺不弱，原财官印俱无损伤，日干之气，又得中和，并无起发可观，此是平常人也。（Line 39）

### 7.2 各格局的病药分析

| 格局 | 病 | 药 |
|---|---|---|
| 正官格 | 官星太重（变为杀） | 行印运/食神制杀 |
| 偏官格 | 七杀无制 | 食神制杀 / 印绶化杀 |
| 财格 | 比劫夺财 | 官杀去比劫 |
| 伤官格 | 伤官见官 | 财印双全以解 |
| 印绶格 | 印绶太重 / 财星破印 | 行比劫运 / 行官杀运 |

### 7.3 病药与"中和"的关系

张楠特别强调病药与中和的辩证关系：

> 盖人之造化，虽贵中和，若一于中和，则安得探其消息，而论其休咎也？

即：贵人之命必有偏颇（病），然后有重药救之，格局方高。完全中和者反为平常人。

## 8. 量化/计分相关原文

### 8.1 得一分三格

> **原文**: 得一分三格，财官印绶全。运中逢克破，一命丧黄泉。进气死不死，退气生不生。（Line 1806）

### 8.2 月气浅深（量化概念）

> **原文**: 法分月气浅深，得令不得令。（Line 1305）

"月气浅深"意味着在月令中有节气深浅的量化概念，如节气前后，气有浅深之不同。

### 8.4 得令与失令的计分逻辑

> **原文**: 甲乙木生丑月，必主光亨。壬癸水值孟秋，终当富贵。（Line 1735）

这种季节→干支配对的模式暗含计分逻辑：

| 天干 | 得令季节 | 失令季节 |

### 8.5 "第一"的量级判定

> 宜邑第一富人也。（Line 67）
> 运登第一。（Line 264）

"第一"在书中出现时，表示在病药配置极佳的情况下所能达到的最高等级。
