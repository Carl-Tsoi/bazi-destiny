## 1. 双轨制理论基调

《三命通会》的核心特征：融合 **古法禄命（纳音/神煞/十二宫）** 与 **今法子平（月令格局/十神喜忌）** 两条理论线，二者并重而非取代。

### 1.1 纳音与神煞体系

#### 1.1.1 六十甲子纳音五行

纳音以 **同类娶妻，隔八生子** 的律吕相生法生成：

```
甲子金(仲) → 娶乙丑 → 隔八下生壬申(孟)
壬申金(孟) → 娶癸酉 → 隔八上生庚辰(季)
庚辰金(季) → 娶辛巳 → 隔八下生戊子(火仲)
...五音一终，复自甲午金之仲，终于癸亥
```

**纳音取象原则**（以金为例）：
- 海中金(甲子乙丑)：气在包藏，有名无形
- 金泊金(壬寅癸卯)：绝地存金，气尚柔弱
- 白蜡金(庚辰辛巳)：金在矿，寄形生养之乡
- 沙中金(甲午乙未)：气已成，混沙而别于沙
- 剑锋金(壬申癸酉)：气盛物极，颖脱锋锐
- 钗钏金(庚戌辛亥)：气藏伏，形体已残

**关键规则**：六十甲子纳音取象对比（对宫阴阳分判）
```
海中<->沙中   水土之辨，阴阳之分
金泊<->剑锋   金木之辨，刚柔之别
白蜡<->钗钏   干巽异方，形色各尽
桑柘<->杨柳   一曲一柔，形质多别
松柏<->石榴   一坚一辛，性味迥异
大林<->平地   一盛一衰
霹雳<->天上   雷霆挥鞭，日月同照
炉中<->山下   火盛木焚，金旺火灭
覆灯<->山头   含光畏风，投光止艮
壁上<->路旁   形分聚散，类别死生
城头<->大驿   东南西北，坤艮正位
沙中<->屋上   干湿互用，变化始终
```

### 1.2 子平格局体系（徐子平论格局）

《三命通会》卷五专论子平格局，以**印、食、官、财四者为纲**。

#### 1.2.1 十神命名体系
```
生我者 → 印绶（荫也，如父母荫庇子孙）
我生者 → 食神（如虫之食物）
克我者 → 官煞（官者管也，煞者害也）
我克者 → 妻财（我受制于人，如人成家得妻内助）
```

#### 1.2.2 格局纲要
- 徐子平只论财、官、印、食，分为六格
- 其余格局不过自此推之
- 八字富贵、贫贱、寿夭、穷通举不外是

#### 1.2.3 正官格核心规则
- 六格之首，止许一位，多则不宜
- 先看月令，然后方看其余
- 大忌：刑冲破害、伤官七煞、贪合忘官、劫财分福
- 正官格要行印乡（逢官看印）
- 官止一二，无财有印，身弱无妨
- 财官满目日主衰弱 → 徒劳无用

#### 1.2.4 偏官格核心规则
- 有制谓之偏官，无制谓之七煞
- 煞不离印，不离煞，煞印相生功名显达
- 身强煞浅，假煞为权；煞重身轻，终身有损
- 煞无刃不显，刃无煞不威
- 时上偏官一位强为佳

#### 1.2.5 印绶格
- 印绶喜官星，逢官则贵
- 印绶怕财星，贪财则坏
- 行官印运便发

#### 1.2.6 财格
- 财喜食神生扶
- 财怕比劫分夺
- 财多身弱 → 富屋贫人

## 2. 标准推断流程SOP

### 2.1 本位论年 → 子平论日的双轨切换

```
// 双轨选择逻辑
IF 使用古法禄命:
    以年为本 → 年柱定格局核心
    遁月从年
ELSE IF 使用子平法:
    以日为主 → 日干为元
    遁时从日
    以月令为提纲
    以年月时合看生克制化，旺相休囚，取立格局
```

**关键原则**：
- 唐李虚中独以日干为主，发前贤所未发，今术家宗之
- 好年不如好月，好月不如好日，好日不如好时 — **但月令同样不可轻**
- 年月日时均重，而时为尤重（人命贵贱寿夭穷通只在生时之辨）

### 2.2 逐层推断流水线（伪代码）

```
FUNCTION 推断八字(年干, 年支, 月干, 月支, 日干, 日支, 时干, 时支):
    // STEP 1: 定三元结构
    天元 = [年干, 月干, 日干, 时干]
    地元 = [年支, 月支, 日支, 时支]
    人元 = 各支藏干(年支, 月支, 日支, 时支)

    // STEP 2: 看岁德（年柱根基）
    年为太岁，主人一生祸福
    日月相生相应 → 根基牢固
    反克冲破 → 本主无寿

    // STEP 3: 看月令格局（核心提纲）
    月为运元
    以月令定正官/偏官/正财/偏财/印绶/食神格局
    先论官星 → 看月令支辰
    次论财印 → 看日干生克

    // STEP 4: 看日主强弱
    日干在月令的旺相休囚死
    地支中有无根气（长生/临官/帝旺）
    天干有无印比生扶
    四柱有无克泄耗

    // STEP 5: 看神煞吉凶
    FOR EACH 神煞 IN [天乙贵人, 禄, 驿马, 羊刃, 空亡, 华盖, 桃花, 三刑]:
        IF 干支组合触发该神煞:
            标记: {
                吉神: 天乙贵人, 禄, 天月德, 学堂词馆
                凶煞: 羊刃, 空亡, 劫煞, 亡神, 三刑, 元辰
                中性: 驿马, 华盖, 桃花
            }

    // STEP 6: 看特殊格局（取代常规格局）
    FOR EACH 奇特局 IN [壬骑龙背, 六阴朝阳, 六乙鼠贵, 飞天禄马, ...]:
        IF 条件全匹配:
            按特格处理，跳过常规格局
            BREAK

    // STEP 7: 看纳音组合
    年柱纳音 → 看生月旺衰
    纳音生克 → 看体用关系
    纳音变化 → 大小强弱之变

    // STEP 8: 看大运顺逆
    IF 阳男 OR 阴女:
        大运顺排（生日后未来节气）
    ELSE:
        大运逆排（生日前过去节气）

    // STEP 9: 看流年太岁
    岁伤日干 → 有祸必轻（上治下顺）
    日犯岁君 → 灾殃必重（下凌上逆）
    岁运并临 → 羊刃七煞为凶，财官印绶为吉
    岁运冲月 → 必祸
    返吟（岁运与日相对）→ 不利六亲
    伏吟（岁运压日）→ 不利六亲

    RETURN 综合论断
```

### 3.1 壬骑龙背格

| 属性 | 内容 |
|------|------|
| **名称** | 壬骑龙背 |
| **出处** | 《喜忌篇》：阳水迭逢辰位，是壬骑龙背之乡 |
| **绝对条件** | ① 日干为壬 ② 日支为辰（壬辰日）③ 地支中辰多见方能冲起戌中财官 |
| **加分条件** | 再得一寅字合住戌为妙 |
| **忌见/破格** | ① 见丁（财）已（官）午（财官）→ 填实不取 ② 柱有己官、戊煞、乙伤、丁合 → 不入格 ③ 怕见北方亥子运 ④ 柱见申子 → 作润下格论，不为骑龙 |
| **系统断语** | 大贵。辰多官爵显，寅多作富家翁。柱中俱有寅辰字，富贵双全在庙堂 |
| **变体** | 壬日坐寅，柱中辰多亦可取此格（壬食甲，甲合己为壬官，甲生丁为壬财） |

## 4. 岁运与原局作用规则

### 4.1 大运基本规则

```
// 大运顺逆规则
IF (阳年男 OR 阴年女):
    大运顺排（以生日后未来节气日时为数）
ELSE:
    大运逆排（以生日前过去节气日时为数）

// 大运作用原则
在干兼用地支之神
在支则弃天干之物
大运重地支（东方、南方、西方、北方之辨）
```

### 4.2 大运吉凶判定

```
// 厄神作用
用神喜生扶 → 欲运生之
用神喜制伏 → 欲运制之
身弱 → 欲运引进旺乡
官 → 欲运生，不欲运伤
煞 → 欲运制，不欲运助
财 → 欲运扶，不欲运劫
印 → 欲运旺，不欲运衰
食 → 欲运生，不欲运枭绝

// 运程阶段吉凶
长生 → 创建作新之事
临官/帝旺 → 兴盛快乐，发福进财
衰/病 → 退败、破财、疾病
死/绝 → 骨肉死丧，自身衰祸
败运 → 落魄懒惰，酒色昏迷
胎库成形/冠带 → 百事得中，安康平易
```

### 4.3 流年太岁规则

```
// 岁君与日主关系
岁伤日干（庚年克甲日）→ 灾轻（上治下顺）
日犯岁君（甲日克戊年）→ 灾重（下凌上逆）
    五阳干犯之重，五阴干犯之轻
    若有天月德救解则无害

// 真太岁与征太岁
真太岁: 生年干支与流年相同 (如甲子年又见甲子年)
    和顺则吉，刑冲破害则凶
征太岁: 日干支冲克太岁，或运干支冲伤太岁
    其年则凶

// 岁运关系
岁运冲月 → 必祸
返吟（岁运与日相对）→ 不利六亲
伏吟（岁运压日）→ 不利六亲
岁运并临 → 羊刃七煞为凶，财官印绶为吉
运克岁 → 主破耗丧事（尤忌），有贵人禄马解之稍吉

// 晦气
日干时干不宜与太岁元天元合 → 合则名晦气
甲合己 → 灾重
己合甲 → 灾轻
岁位近者灾重，远者灾轻
岁在日前五辰遇合 → 太岁入宅，晦气临门
```

### 4.4 小运规则

- 大运司十年之休咎，小运掌一岁之灾祥
- 小运补大运之不足
- 童限未交大运，专用小运
- 大运虽吉，小运不通，不可便言吉利

### 4.5 胎元规则

- 胎月是四柱之根苗
- 日时虽为紧，若不犯破胎月则福多
- 日时之吉而为胎月所犯，则吉亦归无用
- 胎元取法：以当生前三百日为十月之气

### 4.6 伤官见官在岁运中的触发

- 食神格怕枭神夺食
- 财格怕比劫分财
- 印格怕财星坏印

ELSE:
    常规子平格局权重 = 45%

### 5.3 双轨制并行设计

```
// 《三命通会》兼容双轨：古法禄命 + 今法子平

// 轨A：古法禄命（以年为本）
FUNCTION 古法推算(四柱):
    年柱定五行 → 年柱纳音为体
    月柱定旺衰 → 生月对年柱纳音的旺相休囚
    神煞判断 → 以年干、年支查神煞
    纳音生克 → 各柱纳音间的生克制化
    
// 轨B：今法子平（以日为主）
FUNCTION 子平推算(四柱):
    日干为元 → 定十神关系
    月令为纲 → 定格局类型
    日主强弱 → 看地支有无根气
    财官印食 → 看透干通支

// 综合输出（加权平均）
FUNCTION 综合论断(古法结果, 子平结果, 权重配置):
    IF 古法与子平结果一致:
        可高度置信 → 权重翻倍
    IF 古法与子平结论相反:
        降权处理 → 以子平为主(60%)，古法为辅(40%)
        标注"命带矛盾信息，需结合具体情境"
```

### 5.4 关键编程注意事项

1. **特殊奇格优先匹配**：四柱应先检查是否满足特殊格条件，再fallback到常规格局
2. **破格条件必须严格独立**：绝对成立条件与破格条件不能有交集，否则逻辑死循环
3. **使用三元论而非二元论**：天元（干）、地元（支）、人元（藏干）均需独立计算
4. **时辰精度**：时有八刻，初正末气不同，需处理跨时辰边界
5. **十干生死诀硬编码**：甲生亥死午、乙生午死亥、丙生寅死酉、丁生酉死寅、戊生寅死酉、己生酉死寅、庚生巳死子、辛生子死巳、壬生申死卯、癸生卯死申
6. **岁运关系优先**：岁运冲月优先于格局判断，岁运并临优先于大运判断
