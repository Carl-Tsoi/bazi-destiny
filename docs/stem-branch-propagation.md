**文件**: `/tmp/神峰通考.txt`
**行号**: 35-36

```

何以谓之盖头也？如人之一身，独有头为一身为端也。头与面相连，耳目中鼻系焉，
统而言之为之头也。其下若四肢肚腹，稍有不善，可以衣服以饰其不善也，若头之

是五藏六腑也，如肚腹秀气，发出在面头上来，便是英发华出外来，一生富贵贫贱，
只从头面上见得。如八字畏伤官，这伤官藏在内，尚不足畏，如天下透此伤官，便
是头面上已见了，怎能掩饰？凡有所害之物，露出头面，便是动物，就能作害。
凡行运如原八字是乙日干，用丙丁火为伤官，乙日干伤重者，便以庚金官星为病，
若八字上见了庚金，便要丙丁为疾病之神。如早年行壬申癸酉运，便不是好运。
盖因壬癸水盖在申酉头上，是壬癸水盖了头，便不好也。后行甲戌乙亥运，便好了，
是甲乙木盖了头也。又行丙子丁丑动又好，盖行丙丁火盖来克庚也，虽下面地支有
亥子丑水，其水被丙丁盖了头，亦不能为害。又如庚辛日干，喜甲乙丙丁四字为神，
庚辛壬癸四字为病神，行运望见甲乙丙丁数字，盖了头便好。如望见庚辛壬癸数字，
```

**文件**: `/tmp/子平真诠.txt`
**行号**: 91-94

```
盖隔于有所间也，譬如人彼此相好，而有人从中间之，则交必不能成。譬如甲与己合，
```

**原理**: 天干之间的合受两种距离因素限制：(1) 中间有他干阻隔（隔于有所间），则合不成；(2) 位置太远（年干与时干），则只成"半合"，效力仅十之二三。

### 《千里命稿》韦千里 — 紧贴 / 隔位 / 遥隔

**文件**: `/tmp/千里命稿.txt`
**行号**: 2418-2419, 2426-2427, 2467, 2492-2495, 2403-2411

```
（一）如甲年己月。甲己之地位紧贴，合力最重。
```

### 《五行精纪》— 以干为禄 / 以支为命

**文件**: `/tmp/五行精纪.txt`
**行号**: 1191-1193

```
【以干为禄向背定其富贵；以支为命，详逆顺以循环】
王氏注云：如甲禄在寅，或坐禄、向禄者，不富则贵；或破禄、背禄者，不贫则夭。
甲禄在寅，命得寅以坐之，遇丑以向之，逢申以破之，见卯以背之，经云：向禄则生，
背禄则死，故禄前一辰为羊刃，禄后一辰为禄库，此所谓向背定其贫富者欤。十二支曰命，
命之所司者，修短定焉。盖得之以顺者，遇五行生旺，三合六合者也。不顺者逢休囚六害
四衡三刑者也，得此者寿，失此者夭，故曰以支为命，详逆顺以循环。
```

**原理**: 天干决定"禄（官禄）"——朝向或背离禄位决定贫富；地支决定"命（寿命）"——顺逆决定寿夭。这是关于天干地支分管不同领域的核心理论。

## 重要补充发现

### "截脚"一词未见于经典文献

经过对全部十部文献的逐字搜索，"截脚"（severed foot）一词并未出现在任何一部古典命理文献中。这是一个后世的术语发展。古典文献中涵盖的概念通过"坐X绝地"、"干与支同"、"干支相克"、"八字中上四个字是头也，下地支四字是肚腹四肢也"（神峰通考盖头说）等来描述。

### 干支各不相缀

**文件**: `/tmp/五行精纪.txt`
**行号**: 1597, 368

```
干不侵支，则天乃能高；支不犯干，则地乃能卑，五行不相贼，则人乃能顺。
...
干支各不相缀也。
```

**原理**: 理想的干支关系是天干不侵犯地支，地支不冒犯天干，五行不相害。如果干支"不相缀"（不连接），则是一种病理状态。

### 关于干与支关系的语源

**文件**: `/tmp/命理探源.txt`
**行号**: 24-28

```
《五行大义》云：支干者，因五行而立之...蔡邕《月令掌句》云：大挠采五行之情，
占斗机所建，始作甲乙以名之，谓之干。作子丑以名之，谓之支。有事于天则用日，
有事用地则用辰，阴阳之别，故有干支名也。
```

**原理**: 干为天（日），支为地（辰）—— 干支命名源于天文观测，天干代表天象，地支代表地时。
