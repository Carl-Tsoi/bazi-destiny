### 1.2 阴阳干核心差异（算法硬规则）

**规则 1.2.1 — 五阳从气不从势**
```
IF 天干 ∈ {甲,丙,戊,庚,壬}:
    其行为由"气"（月令旺衰、根气深浅）主导
    "势"（财杀数量多寡）对其影响较小
    算法权重：气(0.7) >> 势(0.3)
```

**规则 1.2.2 — 五阴从势无情义**
```
IF 天干 ∈ {乙,丁,己,辛,癸}:
    其行为由"势"（所临五行多寡）主导
    "势盛则从"（木盛从木、火盛从火…）
    算法权重：势(0.7) >> 气(0.3)
```

**穷通宝鉴补充 — 阴阳干性质在四季中的表现：**
- 甲木：春月余寒喜火暖，夏月根乾需水润，秋月宜金（火制金），冬月盘屈要火
- 乙木：春如桃李，夏如禾稼，秋如桐桂，冬如奇葩 — 四季形态各异，用神随季变换
- 丙火：春喜木生(不宜过旺)欲水既济，夏喜水制(免自焚)，秋喜木生(复明之庆)，冬喜木生(绝处逢生)
- 丁火：与丙近似但更依赖甲木，所谓"如有嫡母，可秋可冬"
- 庚金：春需厚土辅(火暖之)，夏喜水盛(制火润土生金)，秋当权喜火煅(成钟鼎材)，冬需火来助土(温养)
- 辛金：春需火暖，夏更柔弱喜水，秋当权喜水泄秀，冬形寒性冷需火助土

### 1.3 天干"活木/死木"判定（穷通宝鉴补充）

### 1.4 十二地支关系动态判定

#### 1.4.1 地支阴阳动静（基础属性）

| 属性 | 地支 | 行为特征 |
|------|------|---------|
| 阳支（动且强） | 子、寅、辰、午、申、戌 | 发速、灾祥显、影响力即时 |
| 阴支（静且专） | 丑、卯、巳、未、酉、亥 | 发缓、否泰经年、影响力延迟 |

**算法规则 1.4.1：**
```
地震影响系数 = IF 阳支 THEN 1.0 ELSE 0.5
时效衰减系数 = IF 阳支 THEN 0  ELSE 1  (经年方显，需年单位累积)
```

#### 1.4.2 生方怕动、库宜开、败地逢冲

**规则 1.4.2 — 三类地支冲的差异化处理：**
```
// 三重地支持遇冲的响应模式
CASE 地支类型 OF
  生方(寅、申、巳、亥):  冲 = 坏（"忌冲动"）→ 优先避免
  四库(辰、戌、丑、未):  冲 = 好（"宜冲开"）→ 库开则五行出
  败地(子、午、卯、酉):  冲 = 视情况而定（有合喜冲、有冲喜合）
END
```

#### 1.4.3 冲的胜负判定（核心算法）

**规则 1.4.3 — "旺者冲衰衰者拔，衰神冲旺旺神发"**
```
// 两柱相冲的胜负判定算法
冲战力评分(支A冲支B):
  // 1. 计算各自综合势力
  势力A = 原局根气(A) + 月令气(A) + 生扶数(A) + 党众数(A)
  势力B = 原局根气(B) + 月令气(B) + 生扶数(B) + 党众数(B)

  // 2. 判定结果
  IF 势力A >> 势力B (旺者冲衰):
    衰者B被拔 → B干支所代表的人事物受损
    拔除指数 = (势力A - 势力B) / 势力B × 100%
    IF 拔除指数 > 阈值(如80%): 彻底拔除（B支完全无用）
    ELSE: 部分损伤

  IF 势力A << 势力B (衰神冲旺):
    旺者B被激发 → B反而得福
    激发指数 = (势力B - 势力A) / 势力B × 100%
    IF 激发指数 > 阈值(如80%): 猛烈爆发

  IF 势力A ≈ 势力B (势均力敌):
    两败俱伤，需要第三方合神来化解
```

#### 1.4.4 暗冲暗会

**规则 1.4.4 — "暗冲暗会尤为喜，彼冲我兮皆冲起"**
```
// 暗冲/暗会门槛条件
IF 柱中明现之某五行 ≥ 3 (多者):
  则以暗的方式冲起/会合其对应之神
  暗冲效力 = 明冲效力的 70%
  但暗冲多吉（"尤为喜"）
```

**彼冲我 / 我冲彼 判定：**
```
// 参考系切换逻辑
以日主为"我"，月令为"彼"
以月令为"我"，年时为"彼"
以四柱为"我"，大运为"彼"
以大运为"我"，流年为"彼"

冲方向 = IF 我支能克彼支 THEN "我冲彼(吉)" ELSE "彼冲我(凶)"
```

## 二、全局动态平衡算法（核心系统）

### 2.1 源流算法 (何处起根源，流到何方住)

**规则 2.1.1 — 源头判定：**
```
源头定位：
  // 不限于当令之神
  // 选取四柱中最旺最多、可作全局核心的五行
  候选 = 各五行总分评分
  源头五行 = max(候选五行)

  源头定位条件：
    - 该五行在四柱中出现次数 ≥ 3
    - 或该五行在月令当权且透干
    - 或该五行虽不当令但党众多（≥4）
```

**规则 2.1.2 — 流向判定：**
```
// 追踪五行的"流"向何处
流向 = 源头所生之五行 (源头→子)
最终归宿 = 流向最终停靠的五行

IF 归宿五行 是 喜神:
  好归路（富贵命） → 评分 + 30
ELSE:
  气泄为乱 → 评分 - 20

IF 途中遇阻节:
  阻神 = 克流向的五行
  阻节严重度 = 阻神势力 - 流向势力
  IF 阻节严重度 > 0:
    断为凶兆，所阻宫位对应六亲/事件受损
```

### 2.2 通关算法（牛郎织女关）

**规则 2.2 — 通关条件与方式：**
```
IF 两种五行相克严重（"天气欲不降，地气欲上升"）:
  // 典型不通关组合
  组合木土 → 需火通关
  组合火金 → 需土通关
  组合土水 → 需金通关
  组合金木 → 需水通关

  // 实际通关检测
  IF 两相克五行之间存在 通关五行:
    IF 通关五行势力 ≥ 被克方势力的 60%:
      通关成功
    ELSE:
      通关失败（"关内有织女，关外有牛郎"隔阻）
  
  // 通关失败时的补偿检测
  补救检测：
    看有无"引用无合之神"或"刑冲所间之物"
    是否岁运能带来通关五行
    能"胜劫占之神"且"能补所缺之物" → 岁运过关
```

### 2.3 清浊判定

**规则 2.3.1 — 清气判定（"一清到底有精神"）：**
```
清格必要条件（任一满足即算清）：
  - 正官格：身旺有财 + 身弱有印 + 无伤官七杀混杂
  - 或有杂神但"循序得所、有安顿"（位置有序不破局）
  - 闲神不来破局（闲神 = 无喜无恶的旁观五行）

清奇评分 = 基础分(60) + (1 - 杂神干扰指数) × 40
清格门槛 = 清奇评分 ≥ 75
```

**规则 2.3.2 — 浊气判定（"满盘浊气令人苦"）：**
```
浊气判定条件（任一即算浊）：
  - 正官格：身弱混七杀 + 混财 + 食神杂之与官不和
  - 印绶杂之不能扶身反与财星相戕
  - 所有五行无法放至合适位置

// 浊气可解？→ "澄浊求清"
IF 有一神有力 或 行运得所:
  可扫浊气 → 仍为富贵命
ELSE:
  贫贱判定
```

### 2.4 真神与假神

**规则 2.4 — "令上寻其聚得真，假神休要乱真神"：**
```
真神判定：
  月令当中当旺之神透出天干 → 为真神
  如木火透干生于寅月（木火当令）→ 真神得用
  IF 真神无伤（不被忌神克）→ 贵命

假神判定：
  IF 真神得令 但 假神（其他五行）得局而党多：
    真假混杂，一生困顿
  IF 假神得令 但 真神得局而党多：
    同上，需要岁运辨胜负

// 暗处寻真——补救规则
IF 月令真神不透（提纲不照）:
  看暗邀（如己土暗邀戊土）、暗化（乙庚暗化）
  运转至真神旺地 → 亦可发福
```

### 2.5 从格与化气算法

**规则 2.5.1 — 真从格（"真从之象有几人"）：**
```
真从格门槛条件（全部满足）：
  1. 日主孤立无气：日主在地支无根（无本气根、也无余气根）
  2. 地支人元无一毫生扶日主之意
  3. 财官强盛（财星或官杀占据全局主导）

  从财：以财星为主，看其流向需要何种辅佐（火/土/金）
  从杀：以七杀为主
  从儿：以食伤为主

  行运规则：若从财，不可行克财之运（如从木财忌金运）
```

**规则 2.5.2 — 假从格：**
```
假从格条件：
  - 满足真从格的大部分条件
  - 但"中有比劫暗生"（日主有暗藏的微根）
  - 岁运若能去其根、助旺从神 → 也可富贵
  - 但"其人不能免祸，或心术不端"
```

**规则 2.5.3 — 真化气（"化得真者只论化"）：**
```
真化气条件（需全部满足）：
  甲己合化土：
    - 日主甲，生于四季（辰戌丑未月）
    - 单遇一位己土在月时相合
    - 有"辰"字（龙以运之）
    - 无壬癸甲乙戊干扰

  丙辛化水 → 冬月为真
  戊癸化火 → 夏月为真
  乙庚化金 → 秋月为真
  丁壬化木 → 春月为真

  化气成立后的规则：
    "既化矣，又论化神"
    如甲己化土后：
      土寒 → 要火暖
      土太旺 → 要水为财、木为官
    再见甲乙不作为争合（因真化如烈女不更二夫）
```

**规则 2.5.4 — 假化气：**
```
假化气条件：
  - 日主孤弱有合神，不得不化
  - 但暗中有扶日主之物
  - 合神虚弱（无龙运之---即无辰字）
  - 岁运扶起合神、制伏忌神 → 亦能发福
  - 但其人"多执滞偏拗，骨肉欠遂"
```

### 2.6 两气成象格 / 独象 / 全象

**规则 2.6.1 — 两气合而成象：**
```
两气成象硬性条件：
  天干一气 + 地支一气
  例：天干皆木 + 地支皆火（木火通明）
  或：天干皆火 + 地支皆木

  象不可破：见金水则破
  存活条件：不得混杂其他五行
```

**规则 2.6.2 — 独象与全象：**
```
独象（一者为独）：
  如曲直格（纯木）、炎上格（纯火）
  行运喜化地（所生之五行）
  化神要昌旺

全象（三者为一）：
  如伤官生财格
  行运喜财地
  财神要旺
```

**规则 2.6.3 — 形全与形缺（损补规则）：**
```
形全（当令旺而成形）→ 宜损其有余
  甲木生于寅卯辰月 → 形全 → 宜火泄/金克/土耗
  丙火生于巳午未月 → 形全 → 宜水制/金耗/土泄

形缺（失令衰弱）→ 宜补其不足
  戊土生于寅卯辰月 → 形缺 → 宜火生/土助
```

### 2.7 方局混判

**规则 2.7.1 — 方与局不混：**
```
// 方 = 三会（寅卯辰/巳午未/申酉戌/亥子丑）
// 局 = 三合（亥卯未/寅午戌/巳酉丑/申子辰）

IF 柱中有"方":
  不可再混入"局"（如寅卯辰方，不可再搭亥或卯或未）
  混局后果 → 力量过强，行运难两利

IF 方局齐全:
  须天干顺排有序 → 方为吉
  行运不可背之
```

## 三、调候·寒暖燥湿状态机

### 3.1 寒暖判定

**规则 3.1 — "天道有寒暖，发育万物"**
```
// 基础冷暖赋值（地支加权）
寒因子：
  阴支(丑卯巳未酉亥) → +1 寒
  西北地支(申酉戌亥子丑) → +1.5 寒
  金水干支(庚辛壬癸申酉亥子) → +2 寒

暖因子：
  阳支(子寅辰午申戌) → +1 暖
  东南地支(寅卯辰巳午未) → +1.5 暖
  木火干支(甲乙丙丁寅卯巳午) → +2 暖

判定：
  净寒暖值 = ∑暖因子 - ∑寒因子
  
  IF 净寒暖值 > 阈值（如 +4）: 炎燥 → 启动调候（需水）
  IF 净寒暖值 < 阈值（如 -4）: 寒冻 → 启动调候（需火）
  IF 中间值: 正常 → 直接走抑扶
```

### 3.2 燥湿判定

**规则 3.2 — "地道有燥湿，生成品汇"**

| 燥湿类型 | 典型组合 | 后果 | 调候策略 |
|---------|---------|------|---------|
| 过湿 | 水有金生 + 遇寒土（丑辰） | 滞而无成 | 需火燥土（戌未） |
| 过燥 | 火有木生 + 遇暖土（戌未） | 烈而有祸 | 需水湿土（丑辰） |
| 火炎土燥 | 丙戊旺 + 无水 | 生机焦枯 | 水为第一用神 |
| 金寒水冷 | 庚辛壬癸 + 冬月 | 冰结冻滞 | 火为第一用神 |

### 3.3 调候与抑扶冲突的取舍规则

**规则 3.3 — "寒之甚，暖之至，必无好处"**
```
// 冲突处理优先级
IF 调候需求等级 == 紧急（寒暖因子差 > 6）:
  调候用神优先级 >> 抑扶用神
  // 解释：命局已处极端，不调候则无生机
  // 如：冬月金寒水冷，即使日主弱也要先用火暖

IF 调候需求等级 == 中等（3 < 差 <= 6）:
  调候与抑扶并重
  // 优先选择既调候又抑扶的干支
  // 如果无两全者，调候略优先

IF 调候需求等级 == 轻微（差 <= 3）:
  抑扶用神优先级 >> 调候用神
  // 正常以旺衰生克为主
```

**穷通宝鉴补充 — 四季调候要点：**
```
春木：余寒犹存，喜火暖→敷畅，但初春不宜水盛（阴浓湿重根损枝萎）
夏木：根乾叶枯，需水滋润（第一用神），切忌火旺
秋木：初秋喜水（水土相资），仲秋喜金（死木得金而造），暮秋喜火（火炎则木实）
```

## 四、岁运战局判定

### 4.1 基础框架（"休囚系乎运，尤系乎岁"）

**规则 4.1 — 大运与流年权重：**
```
// 概念模型
日主 = 我（本体）
局中之神 = 所带人马工具（舟马引从）
大运 = 所到之地（环境）→ 重地支，但不可无视天干
太岁 = 所遇之人（事件）→ 重天干，但不可无视地支

// 作用关系
大运天干 vs 流年天干 → 战/冲/和/好判定
大运地支 vs 流年地支 → 战/冲/和/好判定
与原局交互 → 整体动态
```

### 4.2 战的判定

**规则 4.2 — "何为战"**
```
运伐岁（大运克流年）：
  丙运 庚年 → 丙克庚
  IF 日主喜庚:
    需丙被降伏（"得戊得丙者吉"）
  IF 日主喜丙:
    岁不降运 → 需戊己和为妙

岁伐运（流年克大运）：
  庚运 丙年 → 丙克庚
  IF 日主喜庚:
    需戊己和丙者吉
  IF 日主喜丙:
    运不降岁 → 不可用戊己泄丙助庚
```

### 4.3 冲的判定

**规则 4.3 — "何为冲"**
```
运冲岁（大运地支冲流年地支）：
  子运 午年
  IF 日主喜子:
    要助子；年干遇制午之神
    IF 午之党多 + 干头遇戊甲 → 必凶

岁冲运（流年地支冲大运地支）：
  午运 子年
  IF 日主喜午:
    子党多 + 干头助子 → 必凶
  IF 日主喜子:
    午党少 + 干头助子 → 必吉
  IF 午重子轻:
    岁不降 → 亦无咎
```

### 4.4 和与好的判定

**规则 4.4：**
```
和（合化为吉）：
  乙运庚年（乙庚合）：
    IF 日主喜金 → 吉
    IF 日主喜木 → 不吉

  子运丑年（子丑合）：
    IF 日主喜土 → 吉
    IF 日主喜水 → 不吉

好（同气相助）：
  庚运辛年 → 好（皆金，喜阳则庚好，喜阴则辛好）
  申支酉年 → 好（皆金）
  // 同类五行相扶
```

### 4.5 天战 vs 地战

**规则 4.5 — "天战犹自可，地战急如火"**
```
天战（天干相战）：
  庚乙 → 天战
  IF 地支顺遂（不受损）→ 无害

地战（地支相战）：
  寅申 或 卯酉 对冲 → 地战
  地战后果："天干不能为力，其势速凶"
  原因：天主动、地主静

天地交战（天干地支同时战）：
  → 必凶无疑（极高风险）

// 化解条件
IF 遇岁运合之会之:
  视胜负，有"可存可发"的可能

IF 有一冲两冲 且 有一合神有力 或 库神/贵神:
  可收其动气，息其争气 → 亦有可能变吉
```

### 4.6 原局崩盘判定（反吟伏吟·战局不可调和）

**组合崩盘条件（危险信号）：**
```
大凶预警(满足任意 2 条即标记为高风险):

  条件A — 反吟：
    如大运天干地支与年柱/月柱/日柱/时柱完全相克（天地同克）
    大运甲申，遇年柱庚寅（甲庚冲、寅申冲）→ 天地全冲

  条件B — 伏吟：
    大运/流年干支与原局某柱完全相同
    如年柱庚申遇庚申大运

  条件C — 战局不可调和：
    喜用神在岁运中被彻底冲克
    同时无合神/库神/贵神以安顿

  条件D — 从格破格：
    从财遇克财之大运流年 → 从格崩盘
    从杀遇生扶日主之大运流年 → 破格大凶

  条件E — "原居崩盘"：
    日主唯一之根被冲拔
    全局气势转逆（原本顺畅的气势被拦腰切断）

```

## 五、面向软件算法引擎的设计建议

### 5.1 权重打分制的核心架构

建议采用 **"五行势力评分 + 动态调节因子"** 双引擎架构：

```
// 核心数据结构
struct 五行系统 {
  // 基础权重
  木: float, 火: float, 土: float, 金: float, 水: float

  // 动态调节因子
  月令因子: float    // 当令×1.5，休囚×0.5
  透干因子: float    // 透干×1.3
  根气因子: float    // 本气根×1.5，中气根×1.0，余气根×0.5
  党众因子: float    // 每个同五行干支×0.3
  方局因子: float    // 三会局×1.5，三合局×1.3
  节气因子: float    // 根据出生节气前后微调

  // 计算函数
  func 综合势力(五行) -> float:
    return 基础权重 × 月令因子 × 透干因子 × (1 + 根气因子 + 党众因子 + 方局因子)
}
```

### 5.2 网络流模型模拟"五行流通"

建议用**有向图网络流**模拟五行流通：

```
// 网络流设计
节点: 天干10 + 地支12 + 四柱4个位置 = 26个基础节点
边: 生克关系（生→ +1，克→ -1）

// 源头—流向算法
1. 找到势力评分最高的五行 → 源头节点
2. 从源头沿"生"的方向传播
   木(60%) → 火 → 土 → 金 → 水 → 木
3. 记录每步的流量衰减（每经过一个节点衰减15%）
4. 最终流量沉淀节点 → "归宿"
5. IF 归宿 == 喜神 → 流通成功
   IF 中途阻断（克方势力大于被克方）→ 流通受阻

//  通关检测算法
IF 某边流量受阻 (流量 < 阈值):
  检查中间是否存在通关五行节点
  IF 有 且 该节点势力 >= 阻断方势力的60%:
    自动打通 ("关通矣")
```

### 5.3 从格/化气的状态机设计

```
enum 状态类型 {
  正格,       // 正常抑扶
  真从格,     // 日主无根，全从于财杀/食伤
  假从格,     // 有微根但不足以自立
  真化气,     // 合化成功
  假化气,     // 合化不完全
  专旺格,     // 独象（曲直仁寿、炎上等）
  两气成象,    // 两种五行均分
}

状态转换规则：
  正格 → 真从格:
    IF 日主根 == 0 AND 财杀综合势力 / 日主势力 >= 5.0

  真从格 → 假从格:
    IF 日主根在岁运中出现 (临时反局)
    → 破格，可能大凶

  正格 → 真化气:
    IF 合化条件满足 (见2.5.3规则)
    切换以化神为核心重新计算
```

### 5.4 八字的"全局健康度"评估函数

建议引入**综合性指标**作为快速筛选用：

```python
def 全局健康度(八字):
  评分 = 0

  # 1. 中和度 (0-30分)
  五行方差 = var(五行势力分布)
  评分 += max(0, 30 - 五行方差 * 系数)

  # 2. 流通度 (0-25分)
  if 源头→归宿路径畅通:
    评分 += 25 * (1 - 衰减系数)
  else:
    评分 += 5  # 仅基础分

  # 3. 清纯度 (0-20分)
  评分 += 20 * (1 - 杂神干扰指数)

  # 4. 调候度 (0-15分)
  if 调候需求等级 == "紧急" and 有调候神:
    评分 += 15
  elif 调候需求等级 == "中等" and 有调候神:
    评分 += 10

  # 5. 真神得用 (0-10分)
  if 真神不受伤:
    评分 += 10

  return 评分  # 满分100
```

### 5.5 岁运战局的响应式架构

建议使用**事件驱动的响应模式**：

```
EVENT 岁运到达:
  1. 计算大运与原局的交互
  2. 计算流年与原局的交互
  3. 计算流年与大运的交互（战/冲/和/好）

  响应规则：
  CASE 流年大运关系:
    战 → 触发"战局分析函数"
    冲 → 触发"冲之力计算" (参考规则4.3)
    和 → 触发"合化检测"
    好 → 触发"扶助增益计算"

  // 关键：动态重算各五行综合势力
  // 因为岁运会改变原局的根气、党众关系
  // 可能导致正格→从格，或从格→破格

  重点关注事件：
  - "旺者冲衰衰者拔"：原局衰神被岁运冲克
  - "衰神冲旺旺神发"：原局旺神被岁运轻微冲激
  - "地战" 触发时即时标记高风险
  - 日主唯一之根被冲 → 预警"根拔"
```
